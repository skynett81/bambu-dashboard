# 电商

电商模块为您提供完整的客户、订单和开票管理系统——非常适合专业或半专业销售 3D 打印品的用户。

访问地址：**https://localhost:3443/#orders**

:::danger 需要电商许可证
电商模块需要有效的许可证。许可证**只能通过 [geektech.no](https://geektech.no) 购买**。没有有效许可证，该模块将被锁定，无法使用。
:::

## 许可证 — 购买与激活

### 购买许可证

1. 访问 **[geektech.no](https://geektech.no)** 并创建账户
2. 选择 **Bambu Dashboard — 电商许可证**
3. 选择许可证类型：

| 许可证类型 | 说明 | 打印机数量 |
|---|---|---|
| **爱好者** | 单台打印机，个人使用和小额销售 | 1 |
| **专业版** | 最多 5 台打印机，商业使用 | 1–5 |
| **企业版** | 无限台打印机，全面支持 | 无限 |

4. 完成付款
5. 您将通过邮件收到一个**许可证密钥**

### 激活许可证

1. 在仪表板中进入**设置 → 电商**
2. 填写以下字段：

| 字段 | 说明 | 必填 |
|------|------|------|
| **许可证密钥** | 来自 geektech.no 的 32 位十六进制密钥 | ✅ 是 |
| **电子邮件地址** | 购买时使用的邮件 | ✅ 是 |
| **域名** | 仪表板运行所在的域名（不含 https://） | 推荐 |
| **电话** | 联系电话（含国家代码，例如 +86） | 可选 |

### 许可证类型 — 标识符绑定

geektech.no 将许可证绑定到一个或多个标识符：

| 类型 | 验证对象 | 使用场景 |
|------|---------|---------|
| **域名** | 域名（例如 `dashboard.company.cn`） | 具有专属域名的固定服务器 |
| **IP** | 公网 IP 地址 | 无域名、固定 IP 的服务器 |
| **MAC** | 网卡 MAC 地址 | 硬件绑定 |
| **IP + MAC** | IP 和 MAC 必须同时匹配 | 最高安全性 |

:::info 自动识别
仪表板在每次验证时会自动发送服务器的 IP 地址和 MAC 地址。您无需手动填写——geektech.no 将在首次激活时注册这些信息。
:::

可以允许多个 IP 地址和 MAC 地址（在 geektech.no 管理后台每行一个）。适用于拥有多个网卡或动态 IP 的服务器。

3. 点击**激活许可证**
4. 仪表板向 geektech.no 发送激活请求
5. 成功激活后将显示：
   - **许可证类型**（爱好者 / 专业版 / 企业版）
   - **到期日期**
   - **最大打印机数量**
   - **许可证持有人**
   - **实例 ID**（您的安装唯一标识）

:::warning 许可证密钥与域名和安装绑定
密钥针对特定的 Bambu Dashboard 安装和域名激活。如需以下操作，请联系 [geektech.no](https://geektech.no) 支持：
- 将许可证迁移到新服务器
- 更改域名
- 增加打印机数量
:::

### 许可证验证

许可证通过 geektech.no 进行认证和同步：

- **启动时验证** — 许可证自动检查
- **持续验证** — 每 24 小时对 geektech.no 重新验证
- **离线模式** — 网络中断时，使用缓存验证，许可证可离线使用最多 **7 天**
- **许可证过期** → 模块锁定，但现有数据（订单、客户）保留
- **PIN 码** — geektech.no 可通过 PIN 系统锁定/解锁许可证
- **续期** — 通过 **[geektech.no](https://geektech.no)** → 我的许可证 → 续期

### 许可证类型与限制

| 计划 | 打印机 | 平台 | 费率 | 价格 |
|------|--------|------|------|------|
| **爱好者** | 1 | 1（Shopify 或 WooCommerce） | 5% | 见 geektech.no |
| **专业版** | 1–5 | 全部 | 5% | 见 geektech.no |
| **企业版** | 无限 | 全部 + API | 3% | 见 geektech.no |

## 客户

### 创建客户

1. 进入**电商 → 客户**
2. 点击**新客户**
3. 填写：
   - **姓名 / 公司名称**
   - **联系人**（对于企业客户）
   - **电子邮件地址**
   - **电话**
   - **地址**（账单地址）
   - **公司注册号**（可选，用于增值税注册企业）
   - **备注** — 内部备注
4. 点击**创建**

### 客户列表

客户列表显示：
- 姓名和联系信息
- 总订单数
- 总销售额
- 最近订单日期
- 状态（活跃 / 非活跃）

点击客户查看所有订单和开票历史。

## 订单管理

### 创建订单

1. 进入**电商 → 订单**
2. 点击**新订单**
3. 从列表中选择**客户**
4. 添加订单项：
   - 从文件库选择文件/模型，或添加自由文本项目
   - 设置数量和单价
   - 如果关联到项目，系统自动计算费用
5. 设置**预计交货日期**
6. 点击**创建订单**

### 订单状态

| 状态 | 说明 |
|---|---|
| 询价中 | 已收到询价，尚未确认 |
| 已确认 | 客户已确认 |
| 生产中 | 打印进行中 |
| 待交付 | 已完成，等待取货/发货 |
| 已交付 | 订单完成 |
| 已取消 | 被客户或您取消 |

点击订单 → **更改状态**以更新状态。

### 将打印关联到订单

1. 打开订单
2. 点击**关联打印**
3. 从历史记录中选择打印（支持多选）
4. 费用数据自动从打印历史中获取

## 开票

完整的开票文档请参见[项目 → 开票](../funksjoner/projects#fakturering)。

可直接从订单生成发票：

1. 打开订单
2. 点击**生成发票**
3. 检查金额和税率
4. 下载 PDF 或发送到客户邮箱

### 发票编号系列

在**设置 → 电商**中设置发票编号系列：
- **前缀**：例如 `2026-`
- **起始编号**：例如 `1001`
- 发票编号按升序自动分配

## 报告和费用

### 费用报告

系统追踪所有交易费用：
- 在**电商 → 费用**下查看费用
- 将费用标记为已报告以用于会计目的
- 按周期导出费用摘要

### 统计数据

在**电商 → 统计数据**中：
- 月销售额（柱状图）
- 按销售额排名的顶级客户
- 最畅销的模型/材料
- 平均订单规模

导出为 CSV 以用于会计系统。
