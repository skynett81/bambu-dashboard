# PWA（渐进式网页应用）

Bambu Dashboard 可作为渐进式网页应用（PWA）安装——无需应用商店，直接从浏览器获得类原生应用体验。您可获得更快的访问速度、后台推送通知以及有限的离线功能。

## 安装为应用

### 桌面端（Chrome / Edge / Chromium）

1. 在浏览器中打开 `https://localhost:3443`
2. 在地址栏中查找**安装**图标（带屏幕图标的向下箭头）
3. 点击该图标
4. 在对话框中点击**安装**
5. Bambu Dashboard 将以独立窗口打开，无浏览器 UI

或者：点击三点菜单（⋮）→ **安装 Bambu Dashboard...**

### 桌面端（Firefox）

Firefox 不直接支持完整的 PWA 安装。建议使用 Chrome 或 Edge 以获得最佳体验。

### 移动端（Android – Chrome）

1. 在 Chrome 中打开 **https://您的服务器IP:3443**
2. 点击三点菜单 → **添加到主屏幕**
3. 为应用命名，点击**添加**
4. 图标将出现在主屏幕——应用全屏打开，无浏览器 UI

### 移动端（iOS – Safari）

1. 在 Safari 中打开 **https://您的服务器IP:3443**
2. 点击**分享**图标（带向上箭头的方块）
3. 向下滚动，选择**添加到主屏幕**
4. 点击**添加**

:::warning iOS 限制
iOS 对 PWA 的支持有限。推送通知仅在 iOS 16.4 及更高版本上有效。离线模式功能受限。
:::

## 离线模式

PWA 缓存必要资源以支持有限的离线使用：

| 功能 | 离线可用 |
|---|---|
| 最近已知的打印机状态 | ✅（来自缓存） |
| 打印历史 | ✅（来自缓存） |
| 耗材库存 | ✅（来自缓存） |
| 实时状态（MQTT） | 需要连接 |
| 摄像头画面 | 需要连接 |
| 向打印机发送指令 | 需要连接 |

离线状态下顶部显示横幅：「连接已断开——显示最后已知数据」。

## 后台推送通知

即使应用未打开，PWA 也可以发送推送通知：

1. 打开 PWA
2. 进入**设置 → 通知 → 浏览器推送**
3. 点击**启用推送通知**
4. 接受权限请求
5. 通知将发送到操作系统的通知中心

推送通知适用于[通知](../funksjoner/notifications)中配置的所有事件。

:::info Service Worker
推送通知需要浏览器在后台运行（不能完全关闭系统）。PWA 使用 Service Worker 接收通知。
:::

## 应用图标和外观

PWA 自动使用 Bambu Dashboard 图标。如需自定义：

1. 进入**设置 → 系统 → PWA**
2. 上传自定义图标（最小 512×512 px PNG）
3. 设置**应用名称**和**简称**（显示在移动端图标下方）
4. 选择移动端状态栏的**主题色**

## 更新 PWA

服务器更新时 PWA 自动更新：

- 显示低调横幅：「新版本可用——点击以更新」
- 点击横幅加载新版本
- 无需手动重新安装
