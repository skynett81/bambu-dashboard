# 特殊打印板（PEO、PEY、PEX）

除了四种标准打印板外，Bambu Lab 和第三方制造商还提供具有独特涂层和表面图案的特殊打印板。这些打印板主要用于**打印件底面外观**是成品重要组成部分的情况。

所有特殊打印板均使用相同的弹簧钢磁性固定系统，并与所有 Bambu Lab 打印机兼容。

---

## PEO — 全息/六边形图案

PEO 打印板在涂层中蚀刻了独特的**六边形图案**。该图案为打印件底面提供几何蜂窝外观，在装饰品中非常受欢迎。

### 特征

- **表面：** 六边形图案 — 底面几何蜂窝纹理
- **效果：** 在光线下呈现全息/金属光泽
- **最高床温：** 80 °C
- **涂层：** PEO（聚环氧乙烷基特殊涂层）

### 最适合的材料

| 材料 | 适合性 | 胶棒 |
|------|-------|------|
| PLA | ★★★★★ | 不需要 |
| PLA+ | ★★★★★ | 不需要 |
| PLA Silk | ★★★★★ | 不需要 |
| PETG | ★★★★ | 少量（可选） |
| TPU | ★★★ | 不需要 |

### 用途

- 装饰人偶和雕塑
- 珠宝和配件
- 书立和微缩模型
- 底面可见的展示模型

:::warning 不适合工程材料
ABS、ASA、PA 和 PC 需要比 PEO 打印板承受能力更高的床温。这些材料请使用工程板或高温板。
:::

## PEY — 闪光/星星效果

PEY 打印板为打印件底面提供**闪光或星星状**图案。细腻的闪光图案从多个角度反射光线，创造出独特而喜庆的外观。

### 特征

- **表面：** 细腻闪光图案 — 星星或钻石效果
- **效果：** 在光线下闪亮反光的底面
- **最高床温：** 80 °C
- **涂层：** PEY（具有结构化表面的特殊涂层）

### 最适合的材料

| 材料 | 适合性 | 胶棒 |
|------|-------|------|
| PLA | ★★★★★ | 不需要 |
| PLA Silk | ★★★★★ | 不需要 |
| PLA+ | ★★★★★ | 不需要 |
| PETG | ★★★★ | 少量（可选） |
| TPU | ★★★ | 不需要 |

### 用途

- 珠宝、手链和装饰品
- 派对用品和道具
- 装饰书立和礼品卡
- 角色扮演和舞台道具

:::danger 请勿在 PEY 上使用胶棒
胶棒会填充闪光图案并永久损坏装饰效果。请始终选择无需粘合剂即可自然附着在类 PEI 表面（PLA、PETG）的材料。
:::

## PET — 碳纤维外观

PET 打印板具有模仿真实碳纤维复合材料外观的**斜纹编织图案**。在美观性很重要的 RC 零件、无人机外壳和技术道具中非常受欢迎。

### 特征

- **表面：** 斜纹编织图案 — 碳纤维外观
- **效果：** 底面的技术、工业美学
- **最高床温：** 90 °C
- **涂层：** PET 特殊涂层

### 最适合的材料

| 材料 | 适合性 | 胶棒 |
|------|-------|------|
| PLA | ★★★★★ | 不需要 |
| PLA-CF | ★★★★★ | 不需要 |
| PETG | ★★★★★ | 不需要 |
| PETG-CF | ★★★★ | 少量（可选） |
| TPU | ★★★★ | 不需要 |

### 用途

- RC 车身和无人机外壳
- 手机壳和电子设备道具
- 需要看起来"专业"的原型
- 无需真正 CF 成本即可获得碳纤维美学的模型

:::tip PLA-CF 和 PETG-CF 效果最佳
在 PET 打印板上使用碳纤维涂层耗材（PLA-CF、PETG-CF）时，碳纤维美学得到增强 — 涂层和耗材在视觉上相互补充。
:::

## PEX — 高温光滑

PEX 是**光滑 PEI 的高温变体** — 想想 Cool Plate，但专为 PC 和 PAHT 等苛刻材料设计。具有光亮底面和卓越热稳定性，极为耐用。

### 特征

- **表面：** 光滑/smooth — 提供像 Cool Plate 一样光亮的镜面底面
- **最高床温：** 135 °C（所有标准打印板中最高）
- **涂层：** PEX 特殊涂层，热稳定且极为耐磨
- **取下：** 冷却时打印件像标准光滑 PEI 一样脱落

### 最适合的材料

| 材料 | 床温 | 胶棒 |
|------|------|------|
| PC | 100–120 °C | 是（推荐） |
| PAHT-CF | 100–110 °C | 是（必须） |
| PA-CF | 90–100 °C | 是 |
| ABS | 90–110 °C | 是 |
| ASA | 90–110 °C | 是 |
| PLA | 35–45 °C | 否 |
| PETG | 70–85 °C | 是（隔离层） |

### 相对标准高温板的优势

- 更高的最高床温（135 °C vs. 120 °C）
- 高温板提供光滑但略哑光面，而 PEX 提供光滑底面
- 更长使用寿命 — PEX 涂层更硬更耐用
- 重复热循环时更好的尺寸稳定性

:::warning PEX 不是装饰板的替代品
PEX 是技术高性能产品。光滑表面是功能性的，而不是像 PEO/PEY 那样的装饰性。选择 PEX 用于材料要求高的打印，而不是图案效果。
:::

## 特殊打印板比较

| 打印板 | 图案 | 最高温度 | 最佳用途 | 使用寿命 |
|--------|------|---------|---------|---------|
| PEO | 六边形/蜂窝 | 80 °C | PLA、PETG（装饰用） | 200–400 次 |
| PEY | 闪光/星星 | 80 °C | PLA、PLA Silk（装饰用） | 200–400 次 |
| PET | 碳纤维编织 | 90 °C | PLA、PETG、PLA-CF | 250–450 次 |
| PEX | 光滑（smooth） | 135 °C | PC、PAHT-CF、ABS | 300–500 次 |

## 特殊打印板使用技巧

### 避免在装饰板上使用胶棒

:::danger 胶棒会损坏 PEO、PEY 和 PET 上的图案
这些打印板的装饰效果来自涂层中的细微结构。胶棒（PVA）会填充这些结构并损坏图案 — 而且**很难从细小凹槽中洗去**。

选择不需要胶棒的材料：PLA、PLA+、PLA Silk、PETG。
:::

### 清洁

- **PEO、PEY、PET：** 用 IPA 和软布轻柔擦拭。避免可能磨损图案的研磨清洁
- **PEX：** 像标准光滑 PEI 一样清洁 — IPA，必要时用水和洗碗液

### 存储

特殊打印板应存储在原包装或板架中，以保护图案免受刮擦和灰尘。装饰图案上的刮痕在成品打印件上可见。

### 兼容性

所有特殊打印板均与以下型号兼容：
- Bambu Lab X1C
- Bambu Lab X1E
- Bambu Lab P1S
- Bambu Lab P1P
- Bambu Lab A1 / A1 Mini

:::tip 订购备用打印板
特殊打印板 — 尤其是 PEO 和 PEY — 是比标准打印板寿命更短的消耗品。建议订购一块备用板，以便在等待交货期间不必中断项目。
:::
