# 特殊材料

除了常见材料外，还有许多用于特定用途的特殊材料 — 从耐 UV 的户外部件到水溶性支撑材料。以下是实用概览。

---

## ASA（丙烯腈苯乙烯丙烯酸酯）

ASA 是户外使用 ABS 的最佳替代品。打印方式与 ABS 几乎相同，但对阳光和天气的耐受性要好得多。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 240–260 °C |
| 床温 | 90–110 °C |
| 腔室温度 | 45–55 °C |
| 部件冷却 | 0–20% |
| 干燥 | 推荐（70 °C / 4–6 小时） |

### 特征

- **耐 UV：** 专门设计用于长期日光照射而不会变黄或开裂
- **热稳定：** 玻璃化转变温度约 100 °C
- **耐冲击：** 耐冲击性优于 ABS
- **需要腔室：** 翘边方式与 ABS 相同 — X1C/P1S 效果最佳

:::tip 户外用 ASA 代替 ABS
部件要在户外（阳光、雨水、霜冻）生存吗？选择 ASA 而不是 ABS。ASA 可以多年使用而无明显退化。ABS 会在数月后开始开裂和变黄。
:::

### 用途
- 户外支架、外壳和固定点
- 汽车刮冰器部件、天线支架
- 花园家具和户外环境
- 建筑外部的标识和分配器

---

## PC（聚碳酸酯）

聚碳酸酯是可以 3D 打印的最强、最耐冲击的塑料之一。它是透明的，可承受极端温度。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 260–310 °C |
| 床温 | 100–120 °C |
| 腔室温度 | 50–70 °C |
| 部件冷却 | 0–20% |
| 干燥 | 必须（80 °C / 8–12 小时） |

:::danger PC 需要全金属热端和高温
PC 在标准 PLA 温度下不会熔化。具有正确喷嘴配置的 Bambu X1C 可以处理 PC。始终检查热端中的 PTFE 组件是否能承受您的温度 — 标准 PTFE 无法连续承受超过 240–250 °C。
:::

### 特征

- **非常耐冲击：** 即使在低温下也能抵抗断裂
- **透明：** 可用于窗户、镜头和光学元件
- **热稳定：** 玻璃化转变温度约 147 °C — 常见材料中最高
- **吸湿性：** 快速吸收湿气 — 始终彻底干燥
- **翘边：** 强烈收缩 — 需要腔室和裙边

### 用途
- 安全面罩和保护盖
- 耐热电气外壳
- 镜头座和光学元件
- 机器人框架和无人机机身

---

## PP（聚丙烯）

聚丙烯是最难打印的材料之一，但提供其他任何塑料材料都无法匹敌的独特特性。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 220–250 °C |
| 床温 | 80–100 °C |
| 部件冷却 | 20–50% |
| 干燥 | 推荐（70 °C / 6 小时） |

### 特征

- **耐化学品：** 耐强酸、碱、酒精和大多数溶剂
- **轻质柔性：** 低密度，承受重复弯曲（活铰链效果）
- **低粘附性：** 对自身和打印板粘附性差 — 这是挑战
- **无毒：** 食品接触安全（取决于颜色和添加剂）

:::warning PP 对所有东西粘附都很差
PP 以不粘打印板而闻名。在工程板上使用 PP 胶带（如 Tesa 胶带或专用 PP 胶带），或使用专为 PP 配制的胶棒。需要 15–20mm 的裙边。
:::

### 用途
- 实验室瓶和化学容器
- 食品储存部件和厨房用具
- 活铰链（能承受数千次开闭循环的盒盖）
- 耐化学品的汽车部件

## PVA（聚乙烯醇）— 水溶性支撑材料

PVA 是专门用作支撑材料的特殊材料。它溶于水并在模型上留下干净的表面。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 180–220 °C |
| 床温 | 35–60 °C |
| 干燥 | 关键（55 °C / 6–8 小时） |

:::danger PVA 极具吸湿性
PVA 比任何其他普通耗材吸收湿气更快。打印前彻底干燥 PVA，并始终存储在带硅胶的密封盒中。湿润的 PVA 会粘在喷嘴中且非常难以清除。
:::

### 使用和溶解

1. 以 PVA 作为支撑材料打印模型（需要多材料打印机 — AMS）
2. 将完成的打印件放入温水（30–40 °C）
3. 浸泡 30–120 分钟，必要时换水
4. 用清水冲洗并晾干

如果可能，**始终为 PVA 使用专用挤出机** — 标准挤出机中的 PVA 残留物可能会毁坏下一次打印。

### 用途
- 无法手动移除的复杂支撑结构
- 不留明显表面印记的内部悬垂支撑
- 具有空腔和内部通道的模型

## HIPS（高冲击聚苯乙烯）— 溶剂溶解性支撑材料

HIPS 是另一种支撑材料，设计用于与 ABS 一起使用。它溶于**柠檬烯**（柑橘类溶剂）。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 220–240 °C |
| 床温 | 90–110 °C |
| 腔室温度 | 45–55 °C |
| 干燥 | 推荐（65 °C / 4–6 小时） |

### 与 ABS 一起使用

HIPS 以与 ABS 相同的温度打印，并与其粘附良好。打印后，将打印件浸入 D-柠檬烯 30–60 分钟即可溶解 HIPS。

:::warning 柠檬烯不是水
D-柠檬烯是从橙皮提取的溶剂。它相对无害，但仍需戴手套在通风房间工作。不要将用过的柠檬烯倒入下水道 — 带到废物处理中心。
:::

### 比较：PVA 对比 HIPS

| 特征 | PVA | HIPS |
|------|-----|------|
| 溶剂 | 水 | D-柠檬烯 |
| 兼容材料 | PLA 兼容 | ABS 兼容 |
| 湿度敏感性 | 极高 | 中等 |
| 成本 | 高 | 中等 |
| 可用性 | 好 | 中等 |

## PVB / Fibersmooth — 乙醇可平滑材料

PVB（聚乙烯醇缩丁醛）是一种独特材料，可以**用乙醇（酒精）平滑** — 就像 ABS 可以用丙酮平滑，但安全得多。

### 设置

| 参数 | 值 |
|------|-----|
| 喷嘴温度 | 190–210 °C |
| 床温 | 35–55 °C |
| 部件冷却 | 80–100% |
| 干燥 | 推荐（55 °C / 4 小时） |

### 乙醇平滑

1. 以标准 PVB 设置打印模型
2. 用刷子涂抹 99% 异丙醇（IPA）或乙醇
3. 干燥 10–15 分钟 — 表面均匀流平
4. 如需更光滑表面则重复
5. 或者：涂抹后放入密封容器 5 分钟进行蒸气处理

:::tip 比丙酮更安全
IPA/乙醇比丙酮处理起来安全得多。闪点更高，蒸气毒性低得多。仍建议良好通风。
:::

### 用途
- 需要光滑表面的人偶和装饰品
- 要展示的原型
- 要涂漆的部件 — 光滑表面获得更好涂漆效果

## 特殊材料推荐打印板

| 材料 | 推荐打印板 | 胶棒？ |
|------|-----------|--------|
| ASA | Engineering Plate / High Temp Plate | 是 |
| PC | High Temp Plate | 是（必须） |
| PP | Engineering Plate + PP 胶带 | PP 专用胶带 |
| PVA | Cool Plate / Textured PEI | 否 |
| HIPS | Engineering Plate / High Temp Plate | 是 |
| PVB | Cool Plate / Textured PEI | 否 |
