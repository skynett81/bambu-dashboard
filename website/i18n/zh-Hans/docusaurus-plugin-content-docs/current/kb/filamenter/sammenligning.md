## 选择正确材料 — 决策指南

不确定该选什么？按这些问题来：

### 需要耐热吗？
**是** → ABS、ASA、PC 或 PA

- 较少热量（约 90 °C 以下）：**ABS** 或 **ASA**
- 很多热量（超过 100 °C）：**PC** 或 **PA**
- 最大耐温性：**PC**（约 120 °C）或 **PA-CF**（约 160 °C）

### 需要柔韧性吗？
**是** → **TPU**

- 非常柔软（类似橡胶）：TPU 85A
- 标准柔性：TPU 95A
- 半柔性：PETG 或 PA

### 要在户外使用吗？
**是** → **ASA** 是明确的选择

ASA 专门为 UV 暴露而开发，在户外优于 ABS。如果 ASA 不可用，PETG 是第二选择。

### 需要最大强度吗？
**是** → **PA-CF** 或 **PC**

- 最强轻质复合材料：**PA-CF**
- 最强纯热塑性塑料：**PC**
- 以较低价格获得良好强度：**PA（尼龙）**

### 尽可能简单的打印？
→ **PLA**

PLA 是最宽容的材料。最低温度，无腔室要求，翘边风险最小。

### 食品接触？
→ **PLA**（有保留条件）

PLA 本身无毒，但：
- 使用不锈钢喷嘴（不用黄铜 — 可能含铅）
- 由于多孔表面，FDM 打印永远不是"食品安全"的 — 细菌可以生长
- 避免苛刻环境（酸、热水、洗碗机）
- PETG 是一次性接触的更好选择

## 邵氏硬度解释

邵氏硬度用于描述弹性体和塑料材料的硬度和刚性。对于 3D 打印，它特别适用于 TPU 和其他柔性耗材。

### 邵氏 A — 柔性材料

邵氏 A 刻度从 0（极软，几乎像凝胶）到 100（极硬橡胶）。超过 90A 的值开始接近刚性塑料材料。

| 邵氏 A 值 | 感知硬度 | 示例 |
|---|---|---|
| 30A | 极软 | 硅胶、果冻 |
| 50A | 非常软 | 软橡胶、耳塞 |
| 70A | 软 | 汽车软管、跑鞋中底 |
| 85A | 中等软 | 自行车轮胎、软 TPU 耗材 |
| 95A | 半硬 | 标准 TPU 耗材 |
| 100A ≈ 55D | 刻度之间的边界 | — |

**TPU 95A** 是 3D 打印的行业标准，在弹性和可打印性之间提供良好平衡。**TPU 85A** 非常软，打印时需要更多耐心。

### 邵氏 D — 刚性材料

邵氏 D 用于更硬的热塑性塑料：

| 材料 | 大约邵氏 D |
|---|---|
| PLA | ~80D |
| PETG | ~70D |
| ABS | ~75D |
| PC | ~80D |
| PA（尼龙） | ~70–75D |

:::tip 不是同一回事
即使数字看起来接近，邵氏 A 95 和邵氏 D 40 也不是同一回事。刻度不同，仅在 100A/55D 边界附近部分重叠。始终检查供应商指示的刻度。
:::

## 温度耐受性 — HDT 和 VST

了解材料开始变形的温度对功能性部件至关重要。使用两种标准测量：

- **HDT（热变形温度）** — 材料在标准化负载下弯曲 0.25mm 的温度。负载下使用温度的衡量标准。
- **VST（维卡软化温度）** — 标准化针沉入材料 1mm 的温度。无负载绝对软化点的衡量标准。

| 材料 | HDT（°C） | VST（°C） | 实用最高温度 |
|---|---|---|---|
| PLA | 52–60 | 55–65 | ~50 °C |
| PETG | 70–80 | 75–85 | ~70 °C |
| ABS | 85–105 | 95–110 | ~90 °C |
| ASA | 90–105 | 95–108 | ~90 °C |
| TPU | 40–70 | 变化 | ~60 °C |
| PA（尼龙） | 70–180 | 180–220 | ~80–160 °C |
| PA-CF | 100–200 | 200–230 | ~100–180 °C |
| PC | 120–140 | 145–160 | ~120 °C |
| PLA-CF | 55–65 | 60–70 | ~55 °C |

:::warning 热环境中的 PLA
夏天车内的 PLA 部件是灾难的根源。停放车辆的仪表板可达 80–90 °C。任何可能暴露在阳光或热量下的东西都要使用 ABS、ASA 或 PETG。
:::

:::info PA 变种具有非常不同的特性
PA 是材料家族，不是单一材料。PA6 的 HDT 较低（~70 °C），而 PA12 和 PA6-CF 可以在 160–200 °C。始终检查您正在使用的特定耗材的数据表。
:::

## 喷嘴要求

### 黄铜喷嘴（标准）

适用于所有**不含**碳纤维或玻璃纤维填充的材料：
- PLA、PETG、ABS、ASA、TPU、PA、PC、PVA
- 黄铜柔软，会被研磨性材料快速磨损

### 硬化钢喷嘴（复合材料必须）

**必须**用于：
- PLA-CF（碳纤维 PLA）
- PETG-CF
- PA-CF
- ABS-GF（玻璃纤维 ABS）
- PPA-CF、PPA-GF
- 所有名称中含有"-CF"、"-GF"、"-HF"或"碳纤维"的耗材

硬化钢的导热性低于黄铜 — 在喷嘴温度上补偿 +5–10 °C。

:::danger 碳纤维耗材会快速损坏黄铜喷嘴
黄铜喷嘴在仅几百克 CF 耗材后就可能明显磨损。结果是逐渐恶化的欠挤出和不精确的尺寸。如果您打印复合材料，请投资硬化钢。
:::
