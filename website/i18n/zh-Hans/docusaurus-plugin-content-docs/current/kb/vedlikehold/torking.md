# 耗材干燥

湿润的耗材是打印质量差最常见且被低估的原因之一。即使看起来干燥的耗材也可能已经吸收了足够的湿气来破坏结果 — 尤其是对于尼龙和 PVA 等材料。

## 为什么要干燥耗材？

许多塑料类型具有**吸湿性** — 它们随时间从空气中吸收湿气。当湿润的耗材通过热喷嘴时，水分突然蒸发并在熔化的塑料中产生微气泡。结果是：

- 打印过程中出现**爆裂声和噼啪声**
- 喷嘴处可见**雾气或蒸汽**
- 无法通过调整消除的**拉丝和毛发**
- **粗糙或粒状表面** — 尤其是在顶层
- 层粘合不良和微裂纹导致的**薄弱部件**
- 通常应该有光泽或丝绸光泽的材料出现**哑光或泡沫状表面**

:::warning 在调整设置之前先干燥耗材
许多人花费数小时调整回抽和温度却看不到改善 — 因为原因是湿润的耗材。在更改打印设置之前，务必干燥耗材并重新测试。
:::

## 哪些材料需要干燥？

所有塑料类型都可能变湿，但吸湿程度差异很大：

| 材料 | 吸湿性 | 干燥温度 | 干燥时间 | 优先级 |
|---|---|---|---|---|
| PLA | 低 | 45–50 °C | 4–6 小时 | 可选 |
| PETG | 中 | 65 °C | 4–6 小时 | 推荐 |
| ABS | 中 | 65–70 °C | 4 小时 | 推荐 |
| TPU | 中 | 50–55 °C | 4–6 小时 | 推荐 |
| ASA | 中 | 65 °C | 4 小时 | 推荐 |
| PC | 高 | 70–80 °C | 6–8 小时 | 必须 |
| PA/尼龙 | 极高 | 70–80 °C | 8–12 小时 | 必须 |
| PA-CF | 极高 | 70–80 °C | 8–12 小时 | 必须 |
| PVA | 极高 | 45–50 °C | 4–6 小时 | 必须 |

:::tip 尼龙和 PVA 至关重要
PA/尼龙和 PVA 可以在正常室内环境中**几小时内**吸收足够的湿气变得无法打印。不要在不立即干燥的情况下打开这些材料的新卷轴 — 始终从密闭盒或干燥箱中打印。
:::

## 湿润耗材的迹象

您不必总是按照表格干燥耗材。学会识别症状：

| 症状 | 湿气？ | 其他可能原因 |
|---|---|---|
| 噼啪/爆裂声 | 是，非常可能 | 喷嘴部分堵塞 |
| 喷嘴处雾气/蒸汽 | 是，几乎确定 | 没有其他原因 |
| 粗糙、粒状表面 | 是，可能 | 温度过低，速度过高 |
| 无法消除的拉丝 | 是，可能 | 回抽设置错误，温度过高 |
| 薄弱、易碎部件 | 是，可能 | 温度过低，填充设置错误 |
| 颜色变化或哑光表面 | 是，可能 | 温度错误，塑料烧焦 |

## 干燥方法

### 耗材干燥箱（推荐）

专用耗材干燥箱是最简单安全的解决方案。它们保持精确温度，让您在整个任务期间直接从干燥箱打印。

流行型号：
- **eSun eBOX** — 价格实惠，可从箱中打印，支持大多数材料
- **Bambu Lab Filament Dryer** — 针对 Bambu AMS 优化，支持高温
- **Polymaker PolyDryer** — 良好的温度计和温度控制
- **Sunlu S2 / S4** — 价格友好，可同时放多个卷轴

操作步骤：
```
1. 将卷轴放入干燥箱
2. 按上表设置温度
3. 将计时器设为推荐时间
4. 等待 — 不要缩短过程
5. 直接从干燥箱打印或立即包装
```

### 食物脱水机

普通食物脱水机作为耗材干燥箱效果出奇地好：

- 购买价格实惠（约 100–200 元起）
- 良好的空气循环
- 许多型号支持高达 70–75 °C 的温度

:::warning 检查脱水机的最高温度
许多廉价食物脱水机的恒温器不精确，可能变化 ±10 °C。对需要高温的 PA 和 PC，用温度计测量实际温度。
:::

### 烤箱

烤箱可在紧急情况下使用，但需要谨慎：

:::danger 永远不要将普通烤箱用于 PLA 超过 60 °C — 它会变形！
PLA 在 55–60 °C 时就开始软化。热烤箱可能会损坏卷轴、熔化核心并使耗材无法使用。除非确认温度精确校准且低于 50 °C，否则永远不要将烤箱用于 PLA。
:::

对于可承受较高温度的材料（ABS、ASA、PA、PC）：
```
1. 将烤箱预热至所需温度
2. 用温度计验证实际温度
3. 将卷轴放在烤架上（不直接放在烤箱底部）
4. 让门微开以排出湿气
5. 首次使用此方法时进行监控
```

### Bambu Lab AMS

Bambu Lab AMS Lite 和 AMS Pro 具有内置干燥功能（低热量 + 空气循环）。这不能替代完整干燥，但可以在打印过程中保持已干燥耗材的干燥状态。

- AMS Lite：被动干燥 — 限制湿气吸收，不主动干燥
- AMS Pro：主动加热 — 可能进行一定程度干燥，但不如专用干燥箱有效

## 耗材存储

干燥后的正确存储与干燥过程本身同样重要：

### 最佳解决方案

1. **带硅胶的干燥柜** — 配备湿度计和干燥剂的专用柜。将湿度稳定保持在低水平（理想情况下低于 20% RH）
2. **真空袋** — 抽出空气并在内部用干燥剂密封。最便宜的长期存储
3. **带干燥剂的密封袋** — 短期间简单有效

### 硅胶和干燥剂

- **蓝/橙硅胶**表示饱和程度 — 颜色改变时更换或再生（在 120 °C 烤箱中烘干）
- **珠状硅胶**比颗粒状更有效
- 耗材制造商的**干燥剂包**可以再生和重复使用

### 存储箱中的湿度计

便宜的数字湿度计显示箱中当前湿度：

| 相对湿度（RH） | 状态 |
|---|---|
| 低于 15% | 理想 |
| 15–30% | 对大多数材料良好 |
| 30–50% | PLA 和 PETG 可接受 |
| 超过 50% | 有问题 — 特别是 PA、PVA、PC |

## 实用技巧

- **在打印前刚刚干燥** — 干燥的耗材在正常室内环境中可能在几天内再次变湿
- 对 PA、PC 和 PVA **从干燥箱打印** — 不要只是干燥后搁置
- **新卷轴 ≠ 干燥卷轴** — 制造商用干燥剂密封，但存储链可能出问题。始终干燥吸湿材料的新卷轴
- **在干燥的卷轴上标记干燥日期**
- 从干燥箱到打印机的**专用 PTFE 管**最大限度地减少打印过程中的暴露
