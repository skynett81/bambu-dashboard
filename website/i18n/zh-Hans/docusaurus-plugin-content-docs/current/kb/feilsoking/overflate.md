# 表面缺陷

3D 打印件的表面揭示了系统内部发生的很多信息。大多数表面缺陷有一到两个明确原因 — 通过正确诊断，修复起来出人意料地简单。

## 快速诊断概览

| 症状 | 最常见原因 | 首要措施 |
|---|---|---|
| Blobs 和 zits | 过挤出、接缝位置 | 调整接缝，校准流量 |
| 可见层纹 | Z轴抖动、层厚过大 | 换更细的层高，检查Z轴 |
| Elephant foot | 第一层太宽 | Elephant foot 补偿 |
| 振纹/鬼影 | 高速时振动 | 降低速度，启用输入整形 |
| 欠挤出 | 喷嘴堵塞，温度过低 | 清洁喷嘴，提高温度 |
| 过挤出 | 流量率过高 | 校准流量率 |
| 表面气泡 | 顶层太少，冷却不足 | 增加顶层数量，增加冷却风扇 |
| 层分离 | 温度过低，冷却过度 | 提高温度，减少冷却风扇 |

## Blobs 和 zits

Blobs 是表面上不规则的块状突起。Zits 是点状突起 — 通常沿接缝线出现。

### 原因

- **过挤出** — 过多塑料被挤出并向侧面溢出
- **接缝位置不当** — 标准"nearest"接缝将所有过渡集中在同一位置
- **回抽问题** — 回抽不足导致喷嘴中压力积聚
- **湿润耗材** — 湿气产生微气泡和滴料

### 解决方案

**Bambu Studio 中的接缝设置：**
```
Bambu Studio → 质量 → Seam position
- Aligned: 所有接缝在同一位置（可见但整洁）
- Nearest: 最近点（随机分散 blobs）
- Back: 对象背面（推荐用于视觉质量）
- Random: 随机分布（最好地隐藏接缝）
```

**流量率校准：**
```
Bambu Studio → 校准 → Flow rate
以 ±2% 步进调整，直到 blobs 消失
```

:::tip 视觉质量将接缝设置为"Back"
将接缝放置在对象背面使其最不明显。结合"Wipe on retract"获得更清洁的接缝收尾。
:::

## 可见层纹

所有 FDM 打印件都有层纹，但在正常打印件上应该一致且几乎不可见。异常的可见性指向具体问题。

### 原因

- **Z 轴抖动** — Z 轴振动或不直，在整个高度产生波状图案
- **层厚过大** — 即使在完美打印上，超过 0.28mm 的层高也很明显
- **温度波动** — 不一致的熔化温度导致层宽变化
- **耗材直径不一致** — 直径变化的廉价耗材

### 解决方案

**Z 轴抖动：**
- 检查丝杆（Z-leadscrew）是否清洁且有润滑
- 检查丝杆是否弯曲（旋转时目视检查）
- 参见 [Z 轴润滑](/docs/kb/vedlikehold/smoring) 维护文章

**层高：**
- 换成 0.12mm 或 0.16mm 获得更均匀的表面
- 记住层高减半会使打印时间加倍

**温度波动：**
- 使用 PID 校准（通过 Bambu Studio 维护菜单可用）
- 避免在打印过程中冷却喷嘴的气流

## Elephant foot

Elephant foot 是指第一层比对象其余部分更宽 — 就像对象在底部"向外展开"一样。

### 原因

- 第一层对打印板挤压过度（Z偏移过紧）
- 床温过高使塑料长时间保持柔软流动状态
- 第一层冷却不足使塑料有更多时间扩散

### 解决方案

**Bambu Studio 中的 Elephant foot compensation：**
```
Bambu Studio → 质量 → Elephant foot compensation
从 0.1–0.2mm 开始，调整直到脚消失
```

**Z 偏移：**
- 重新校准第一层高度
- 每次将 Z 偏移提高 0.05mm，直到脚消失

**床温：**
- 将床温降低 5–10 °C
- 对于 PLA：55 °C 通常足够 — 65 °C 可能导致 elephant foot

:::warning 不要过度补偿
Elephant foot compensation 过高可能会在第一层和其余部分之间产生间隙。以 0.05mm 步进谨慎调整。
:::

## 振纹和鬼影

振纹（也称为"鬼影"或"回声"）是紧接在锋利边缘或角落之后表面出现的波状图案。图案从边缘"回荡"出去。

### 原因

- **振动** — 角落处的快速加速和减速通过框架传递振动
- **速度过高** — 特别是外壁速度超过 100mm/s 会产生明显振纹
- **松动部件** — 松动的料盘、振动的线缆套管或安装不稳的打印机

### 解决方案

**Bambu Lab 输入整形器（Resonance Compensation）：**
```
Bambu Studio → 打印机 → Resonance compensation
Bambu Lab X1C 和 P1S 有内置加速度计并自动校准
```

**降低速度：**
```
外壁：降至 60–80 mm/s
加速度：从标准降至 3000–5000 mm/s²
```

**机械检查：**
- 确认打印机放置在稳定表面上
- 检查料盘在料盘架中是否振动
- 拧紧框架外部面板上所有可访问的螺丝

:::tip X1C 和 P1S 自动校准振纹
Bambu Lab X1C 和 P1S 具有内置加速度计校准，在启动时自动运行。如果振纹在一段时间后出现，请从维护菜单运行"Full calibration"。
:::

## 欠挤出

欠挤出是指打印机挤出塑料过少。结果是薄而弱的墙壁、层间可见孔洞和"参差不齐"的表面。

### 原因

- **喷嘴部分堵塞** — 碳积聚减少流量
- **喷嘴温度过低** — 塑料流动性不足
- **挤出机中磨损的齿轮**没有充分抓住耗材
- **速度过高** — 挤出机跟不上所需流量

### 解决方案

**冷拔（Cold pull）：**
```
1. 将喷嘴加热至 220 °C
2. 手动推入耗材
3. 保持压力同时将喷嘴冷却至 90 °C（PLA）
4. 快速拉出耗材
5. 重复直到拉出的部分干净
```

**温度调整：**
- 将喷嘴温度提高 5–10 °C 并重新测试
- 温度过低是欠挤出的常见原因

**流量率校准：**
```
Bambu Studio → 校准 → Flow rate
逐渐增加，直到欠挤出消失
```

**检查挤出机齿轮：**
- 取出耗材并检查齿轮
- 如果牙齿中有耗材粉末，用小刷子清洁

## 过挤出

过挤出产生过宽的条纹 — 表面看起来松散、有光泽或不均匀，倾向于出现 blobs。

### 原因

- **流量率过高**（EM — Extrusion Multiplier）
- **耗材直径错误** — 使用 1.75mm 配置文件的 2.85mm 耗材会导致大量过挤出
- **喷嘴温度过高**使塑料过于流动

### 解决方案

**流量率校准：**
```
Bambu Studio → 校准 → Flow rate
以 2% 步进降低，直到表面均匀且哑光
```

**验证耗材直径：**
- 沿耗材 5–10 处用游标卡尺测量实际耗材直径
- 超过 0.05mm 的平均偏差表明低质量耗材

## 表面气泡（Pillowing）

Pillowing 是顶层之间有"枕头状"塑料的泡沫状、不均匀顶层。低填充率和顶层太少时尤为明显。

### 原因

- **顶层太少** — 填充上方的塑料落入孔洞
- **冷却不足** — 塑料凝固不够快，无法跨越填充孔
- **填充率过低** — 填充中的大孔难以覆盖

### 解决方案

**增加顶层数量：**
```
Bambu Studio → 质量 → Top shell layers
最少：4 层
均匀表面推荐：5–6 层
```

**增加冷却：**
- PLA 应从第 3 层开始冷却风扇设为 100%
- 冷却不足是 pillowing 最常见的原因

**增加填充率：**
- 如果 pillowing 持续，从 10–15% 增加到 20–25%
- Gyroid 图案提供比线条更均匀的桥接表面

:::tip 熨烫（Ironing）
Bambu Studio 的"ironing"功能额外运行一次喷嘴扫描顶层来平滑表面。在质量 → Ironing 中启用以获得最佳顶层表面处理。
:::

## 层分离（Layer separation / delamination）

层分离是指层与层之间没有正确粘合。在最坏情况下，打印件沿层线裂开。

### 原因

- **喷嘴温度过低** — 塑料没有充分熔入下方层
- **冷却过度** — 塑料在熔合前凝固过快
- **层厚过大** — 超过喷嘴直径的 80% 会导致熔合不良
- **速度过高** — 每毫米路径的熔化时间减少

### 解决方案

**提高温度：**
- 从标准温度试用 +10 °C 并观察
- ABS 和 ASA 特别敏感 — 需要受控腔室加热

**减少冷却：**
- ABS：冷却风扇关闭（0%）
- PETG：最多 20–40%
- PLA：可承受完全冷却，但如果发生层分离则降低

**层厚：**
- 使用最多 75% 的喷嘴直径
- 0.4mm 喷嘴：推荐最大层高为 0.30mm

**检查腔室是否足够热（ABS/ASA/PA/PC）：**
```
Bambu Lab X1C/P1S：让腔室在打印开始前加热至 40–60 °C
— 打印过程中不要打开门
```

## 一般故障排除技巧

1. **每次只改变一件事** — 在每次更改之间用小型校准打印进行测试
2. **先烘干耗材** — 许多表面缺陷实际上是湿度症状
3. **清洁喷嘴** — 部分堵塞会产生难以诊断的不一致表面缺陷
4. **重大调整后从 Bambu Studio 维护菜单运行完整校准**
5. **使用 Bambu Dashboard** 跟踪随时间产生最佳结果的设置
