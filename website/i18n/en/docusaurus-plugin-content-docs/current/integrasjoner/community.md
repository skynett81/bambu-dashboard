# Community Filaments

Community Filaments is a shared database of filament profiles contributed by Bambu Dashboard users from around the world. Find the best settings for your filament, and share your own experiences.

Go to: **https://localhost:3443/#community**

## Browse community profiles

### Search

1. Go to **Community Filaments**
2. Use the search field to find by:
   - Material (PLA, PETG, ABS, PA, TPU, etc.)
   - Vendor (PolyMaker, Bambu Lab, Prusament, Sunlu, etc.)
   - Color name or color hex
   - Property (transparent, matte, silk, glitter, etc.)
3. Filter by **Rating** (highest rated only)
4. Filter by **Printer model** to see profiles optimized for your printer

### Profile cards

Each profile card shows:

| Field | Example |
|---|---|
| Filament name | PolyTerra PLA – Matte Forest Green |
| Material | PLA |
| Vendor | PolyMaker |
| Contributor | username |
| Rating | ⭐⭐⭐⭐½ (4.6 / 5) |
| Number of tests | 23 |
| Recommended temperatures | Nozzle: 215°C / Bed: 55°C |
| Recommended speed | Standard |

Click on the card to see the full profile with all parameters and user comments.

## Importing a profile

1. Open a profile from the community
2. Click **Import to inventory**
3. Choose whether to:
   - **Link to existing spool** — the profile is linked to a registered spool
   - **Create new spool** — creates a new spool entry with profile data
4. Adjust temperature or other settings as needed
5. Click **Import**

The profile is now available in your filament inventory.

## Contributing your own profiles

### Export and sharing

1. Go to **Filament Inventory**
2. Find the spool you want to share
3. Click **Share with community**
4. Fill in:
   - **Vendor name** — exact vendor name
   - **Product name** — exact product name from the packaging
   - **Color name** — official color name and color hex
   - **Comment** — your experiences, tips and tricks
   - **Printer model** — which printer you tested on
5. Click **Submit**

The profile is under review and typically published within 24 hours.

### Contribution guidelines

:::warning Quality requirements
- Only submit real, tested settings — not speculative values
- Use the exact vendor and product name (check the packaging)
- One profile per specific product and printer model
- Avoid duplicate profiles — check if the profile already exists
:::

## Rating profiles

Help the community by rating profiles you have tested:

1. Open a community profile
2. Click **Give rating**
3. Set the number of stars (1–5) and a short comment
4. Click **Submit**

Profiles with high ratings appear at the top of search results.

## Offline use

The community database syncs locally automatically (updated daily). All profiles are available without an internet connection after the first sync.

Download the database manually: **Community Filaments → Settings → Sync now**
