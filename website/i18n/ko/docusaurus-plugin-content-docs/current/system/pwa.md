# PWA (Progressive Web App)

Bambu Dashboard는 Progressive Web App (PWA)으로 설치할 수 있습니다 — 앱 스토어 없이 브라우저에서 바로 앱과 같은 경험을 제공합니다. 더 빠른 접근, 백그라운드 푸시 알림 및 제한적인 오프라인 기능을 제공합니다.

## 앱으로 설치

### 데스크톱 (Chrome / Edge / Chromium)

1. 브라우저에서 `https://localhost:3443`을 엽니다
2. 주소 표시줄에서 **설치** 아이콘을 찾습니다 (화면 아이콘이 있는 아래쪽 화살표)
3. 클릭합니다
4. 대화 상자에서 **설치** 클릭
5. Bambu Dashboard가 브라우저 UI 없이 별도의 창으로 열립니다

또는: 세 점 (⋮) 클릭 → **Bambu Dashboard 설치...**

### 데스크톱 (Firefox)

Firefox는 전체 PWA 설치를 직접 지원하지 않습니다. 최상의 경험을 위해 Chrome 또는 Edge를 사용하세요.

### 모바일 (Android – Chrome)

1. Chrome에서 **https://서버IP:3443**을 엽니다
2. 세 점 → **홈 화면에 추가** 누르기
3. 앱 이름을 지정하고 **추가** 누르기
4. 홈 화면에 아이콘이 표시됩니다 — 브라우저 UI 없이 전체 화면으로 앱이 열립니다

### 모바일 (iOS – Safari)

1. Safari에서 **https://서버IP:3443**을 엽니다
2. **공유** 아이콘 누르기 (위쪽 화살표가 있는 사각형)
3. 아래로 스크롤하여 **홈 화면에 추가** 선택
4. **추가** 누르기

:::warning iOS 제한사항
iOS는 PWA 지원이 제한적입니다. 푸시 알림은 iOS 16.4 이상에서만 작동합니다. 오프라인 모드가 제한적입니다.
:::

## 오프라인 모드

PWA가 제한적인 오프라인 사용을 위해 필요한 리소스를 캐싱합니다:

| 기능 | 오프라인 사용 가능 |
|---|---|
| 마지막으로 알려진 프린터 상태 | ✅ (캐시에서) |
| 프린트 기록 | ✅ (캐시에서) |
| 필라멘트 재고 | ✅ (캐시에서) |
| 실시간 상태 (MQTT) | ❌ 연결 필요 |
| 카메라 스트림 | ❌ 연결 필요 |
| 프린터에 명령 전송 | ❌ 연결 필요 |

오프라인 보기는 상단에 배너를 표시합니다: 「연결이 끊어졌습니다 — 마지막으로 알려진 데이터를 표시하고 있습니다」.

## 백그라운드 푸시 알림

앱이 열려 있지 않아도 PWA가 푸시 알림을 보낼 수 있습니다:

1. PWA 열기
2. **설정 → 알림 → 브라우저 푸시**로 이동합니다
3. **푸시 알림 활성화** 클릭
4. 권한 대화 상자 수락
5. 알림이 운영 체제의 알림 센터에 전달됩니다

푸시 알림은 [알림](../funksjoner/notifications)에서 구성된 모든 이벤트에 대해 작동합니다.

:::info Service Worker
푸시 알림은 브라우저가 백그라운드에서 실행 중이어야 합니다 (전체 시스템 종료 없음). PWA는 수신에 Service Worker를 사용합니다.
:::

## 앱 아이콘 및 외관

PWA는 자동으로 Bambu Dashboard 아이콘을 사용합니다. 사용자 정의하려면:

1. **설정 → 시스템 → PWA**로 이동합니다
2. 사용자 정의 아이콘 업로드 (최소 512×512 px PNG)
3. **앱 이름** 및 **짧은 이름** 입력 (모바일에서 아이콘 아래 표시)
4. 모바일 상태 표시줄의 **테마 색상** 선택

## PWA 업데이트

서버가 업데이트되면 PWA가 자동으로 업데이트됩니다:

- 눈에 띄지 않는 배너 표시: 「새 버전 사용 가능 — 클릭하여 업데이트」
- 배너를 클릭하여 새 버전 로드
- 수동 재설치 불필요
