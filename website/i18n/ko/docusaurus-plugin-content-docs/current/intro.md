# Bambu Dashboard에 오신 것을 환영합니다

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/V7V21NRKM7)

**Bambu Dashboard**는 Bambu Lab 3D 프린터를 위한 셀프 호스팅 완전 기능 제어판입니다. 프린터, 필라멘트 재고, 인쇄 기록 등 모든 것을 하나의 브라우저 탭에서 완전히 파악하고 제어할 수 있습니다.

## Bambu Dashboard란?

Bambu Dashboard는 LAN을 통한 MQTT로 프린터에 직접 연결하며, Bambu Lab 서버에 의존하지 않습니다. 모델 및 인쇄 기록 동기화를 위해 Bambu Cloud에도 연결할 수 있습니다.

### 주요 기능

- **라이브 대시보드** — 실시간 온도, 진행 상황, 카메라, AMS 상태 (LIVE 인디케이터 포함)
- **필라멘트 재고** — AMS 동기화, EXT 스풀 지원, 소재 정보, 플레이트 호환성, 건조 가이드와 함께 모든 스풀 관리
- **필라멘트 트래킹** — 4단계 폴백을 통한 정확한 추적 (AMS 센서 → EXT 추정 → 클라우드 → 시간)
- **소재 가이드** — 15가지 소재의 온도, 플레이트 호환성, 건조, 특성, 팁
- **인쇄 기록** — 모델명, MakerWorld 링크, 필라멘트 사용량, 비용이 포함된 완전한 로그
- **플래너** — 캘린더 보기, 부하 분산 및 필라멘트 확인이 포함된 인쇄 대기열
- **프린터 제어** — 온도, 속도, 팬, G-code 콘솔
- **Print Guard** — xcam + 5개 센서 모니터를 통한 자동 보호
- **비용 추정** — 소재, 전력, 작업, 마모, 마크업, 판매 가격 제안
- **유지보수** — KB 기반 간격, 노즐 수명, 플레이트 수명, 가이드 포함 추적
- **사운드 알람** — 커스텀 사운드 업로드 및 프린터 스피커(M300) 지원이 포함된 9개의 설정 가능한 이벤트
- **활동 로그** — 모든 이벤트(인쇄, 오류, 유지보수, 필라멘트)의 영구 타임라인
- **알림** — 7개 채널 (Telegram, Discord, 이메일, ntfy, Pushover, SMS, webhook)
- **멀티 프린터** — Bambu Lab 전 시리즈 지원
- **17개 언어** — 노르웨이어, 영어, 독일어, 프랑스어, 스페인어, 이탈리아어, 일본어, 한국어, 네덜란드어, 폴란드어, 포르투갈어, 스웨덴어, 터키어, 우크라이나어, 중국어, 체코어, 헝가리어
- **셀프 호스팅** — 클라우드 의존성 없음, 데이터는 내 컴퓨터에

### v1.1.13의 새로운 기능

- **EXT 스풀 감지** — MQTT 매핑 필드를 통한 P2S/A1 지원, 외부 스풀의 필라멘트 사용량을 정확하게 추적
- **필라멘트 소재 데이터베이스** — 15가지 소재, 플레이트 호환성, 건조 가이드, 특성
- **유지보수 패널** — KB 기반 간격, 4가지 새로운 노즐 타입, 문서 링크가 포함된 가이드 탭
- **사운드 알람** — 9개 이벤트, 커스텀 업로드 (MP3/OGG/WAV, 최대 10초), 볼륨 조절, 프린터 스피커
- **활동 로그** — 페이지가 열려 있지 않아도 모든 데이터베이스의 영구 타임라인
- **HMS 오류 코드** — 270개 이상의 코드의 읽기 쉬운 설명
- **완전한 i18n** — 17개 언어로 번역된 2944개의 모든 키
- **자동 빌드 문서** — 설치 및 서버 시작 시 자동으로 문서 빌드

## 지원 프린터

LAN 모드를 지원하는 모든 Bambu Lab 프린터:

- **X1 시리즈**: X1C, X1C Combo, X1E
- **P1 시리즈**: P1S, P1S Combo, P1P
- **P2 시리즈**: P2S, P2S Combo
- **A 시리즈**: A1, A1 Combo, A1 mini
- **H2 시리즈**: H2S, H2D (듀얼 노즐), H2C (툴 체인저, 6 헤드)

## 기능 상세

### 필라멘트 트래킹

대시보드는 4단계 폴백으로 필라멘트 사용량을 자동으로 추적합니다:

1. **AMS 센서 차이** — 가장 정확함, 시작/종료 잔량%를 비교
2. **EXT 직접** — vt_tray가 없는 P2S/A1용, 클라우드 추정 사용
3. **클라우드 추정** — Bambu Cloud 인쇄 작업 데이터에서
4. **시간 추정** — 최종 폴백으로 약 30g/시간

실패한 인쇄 후 오류를 피하기 위해 모든 값은 AMS 센서와 스풀 데이터베이스의 최솟값으로 표시됩니다.

### 소재 가이드

15가지 소재를 포함한 내장 데이터베이스:
- 온도 (노즐, 베드, 챔버)
- 플레이트 호환성 (Cool, Engineering, High Temp, Textured PEI)
- 건조 정보 (온도, 시간, 흡습성)
- 8가지 특성 (강도, 유연성, 내열성, UV, 표면, 사용 편의성)
- 난이도 및 특별 요구 사항 (하드닝 노즐, 인클로저)

### 사운드 알람

다음을 지원하는 9개의 설정 가능한 이벤트:
- **커스텀 사운드 클립** — MP3/OGG/WAV 업로드 (최대 10초, 500 KB)
- **내장 톤** — Web Audio API로 생성된 메탈릭/신스 사운드
- **프린터 스피커** — 프린터 버저에 직접 M300 G-code 멜로디
- **카운트다운** — 인쇄 1분 남았을 때 사운드 알람

### 유지보수

완전한 유지보수 시스템:
- 컴포넌트 추적 (노즐, PTFE 튜브, 로드, 베어링, AMS, 플레이트, 건조)
- 문서에서 가져온 KB 기반 간격
- 타입별 노즐 수명 (황동, 하드닝 강철, HS01)
- 타입별 플레이트 수명 (Cool, Engineering, High Temp, Textured PEI)
- 완전한 문서로의 팁과 링크가 포함된 가이드 탭

## 기술 개요

Bambu Dashboard는 Node.js 22와 바닐라 HTML/CSS/JS로 구축되었습니다 — 무거운 프레임워크 없음, 빌드 단계 없음. 데이터베이스는 Node.js 22에 내장된 SQLite입니다.

- **백엔드**: Node.js 22, npm 패키지 3개만 사용 (mqtt, ws, basic-ftp)
- **프론트엔드**: 바닐라 HTML/CSS/JS, 빌드 단계 없음
- **데이터베이스**: Node.js 22 내장 `--experimental-sqlite`를 통한 SQLite
- **문서**: 17개 언어를 지원하는 Docusaurus, 설치 시 자동 빌드
- **API**: 177개 이상의 엔드포인트, `/api/docs`에 OpenAPI 문서

자세한 내용은 [아키텍처](./avansert/arkitektur)를 참조하세요.
