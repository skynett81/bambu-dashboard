# 표면 결함

3D 출력물의 표면은 시스템 내부에서 일어나고 있는 것에 대해 많은 것을 알려줍니다. 대부분의 표면 결함에는 한두 가지 명확한 원인이 있으며, 올바로 진단하면 놀라울 정도로 간단하게 수정할 수 있습니다.

## 빠른 진단 개요

| 증상 | 가장 일반적인 원인 | 첫 번째 조치 |
|---|---|---|
| Blobs와 zits | 과압출, 시임 배치 | 시임 조정, 유량 보정 |
| 가시적인 레이어 라인 | Z-wobble, 레이어가 너무 두꺼움 | 더 얇은 레이어로 변경, Z축 확인 |
| Elephant foot | 첫 번째 레이어가 너무 넓음 | Elephant foot compensation |
| 링잉/고스팅 | 고속 시 진동 | 속도 낮추기, 입력 셰이퍼 활성화 |
| 언더 압출 | 노즐 막힘, 온도 너무 낮음 | 노즐 청소, 온도 높이기 |
| 오버 압출 | 유량률 너무 높음 | 유량률 보정 |
| 필로잉 | 상단 레이어 너무 적음, 냉각 부족 | 상단 레이어 수 늘리기, 냉각 팬 증가 |
| 레이어 분리 | 온도 너무 낮음, 냉각 과다 | 온도 높이기, 냉각 팬 줄이기 |

## Blobs와 zits

Blobs는 표면의 불규칙한 덩어리입니다. Zits는 점 같은 돌기로, 보통 시임 라인을 따라 나타납니다.

### 원인

- **과압출** — 과도한 플라스틱이 압출되어 옆으로 밀려남
- **시임 배치 불량** — 표준 "nearest" 시임은 모든 전환점을 같은 위치에 집중
- **리트랙 문제** — 불충분한 리트랙으로 노즐 내 압력 축적
- **습기 찬 필라멘트** — 습기가 마이크로 버블과 흘러내림을 만들어냄

### 해결책

**Bambu Studio에서 시임 설정:**
```
Bambu Studio → 품질 → Seam position
- Aligned: 모든 시임을 같은 위치에 (보이지만 정돈됨)
- Nearest: 가장 가까운 점 (blobs를 무작위로 분산)
- Back: 객체 뒤 (시각적 품질에 권장)
- Random: 무작위 분포 (시임을 가장 잘 숨김)
```

**유량률 보정:**
```
Bambu Studio → 보정 → Flow rate
±2% 단계로 조정하여 blobs가 사라질 때까지
```

:::tip 시각적 품질을 위해 "Back"에 시임 설정
객체 뒤쪽에 시임을 배치하여 가장 덜 보이게 합니다. 더 깨끗한 시임 마감을 위해 "Wipe on retract"와 결합하세요.
:::

## 가시적인 레이어 라인

모든 FDM 출력물에는 레이어 라인이 있지만, 일반 출력물에서는 일관되고 거의 보이지 않아야 합니다. 비정상적인 가시성은 구체적인 문제를 가리킵니다.

### 원인

- **Z-wobble** — Z축이 진동하거나 직선이 아니어서 전체 높이에 걸쳐 파형 패턴이 생김
- **레이어가 너무 두꺼움** — 0.28mm 이상의 레이어 높이는 완벽한 출력에서도 눈에 띔
- **온도 변동** — 일관성 없는 용융 온도로 레이어 너비 변화
- **불일치한 필라멘트 직경** — 직경이 변하는 저가 필라멘트

### 해결책

**Z-wobble:**
- 스크루 스핀들(Z-leadscrew)이 깨끗하고 윤활되어 있는지 확인
- 스핀들이 휘어있지 않은지 확인 (회전 시 육안 검사)
- [Z축 윤활](/docs/kb/vedlikehold/smoring) 유지보수 기사 참조

**레이어 높이:**
- 더 균일한 표면을 위해 0.12mm 또는 0.16mm로 변경
- 레이어 높이를 반으로 줄이면 출력 시간이 두 배가 됨을 기억하세요

**온도 변동:**
- PID 보정 사용 (Bambu Studio 유지보수 메뉴에서 이용 가능)
- 출력 중 노즐을 냉각시키는 기류를 피하세요

## Elephant foot

Elephant foot은 첫 번째 레이어가 객체의 나머지 부분보다 넓을 때 발생합니다 — 마치 객체가 바닥에서 "펼쳐지는" 것처럼 보입니다.

### 원인

- 첫 번째 레이어가 플레이트에 너무 세게 눌림 (Z 오프셋이 너무 촘촘함)
- 베드 온도가 너무 높아 플라스틱이 너무 오래 부드럽고 유동적 상태 유지
- 첫 번째 레이어 냉각 불충분으로 플라스틱이 퍼질 시간이 너무 많음

### 해결책

**Bambu Studio에서 Elephant foot compensation:**
```
Bambu Studio → 품질 → Elephant foot compensation
0.1–0.2mm로 시작하여 발이 사라질 때까지 조정
```

**Z 오프셋:**
- 첫 번째 레이어 높이 재보정
- 발이 없어질 때까지 0.05mm씩 Z 오프셋 올리기

**베드 온도:**
- 베드 온도 5–10 °C 낮추기
- PLA의 경우: 55 °C면 충분한 경우가 많음 — 65 °C는 elephant foot을 유발할 수 있음

:::warning 과도한 보정 금지
Elephant foot compensation이 너무 높으면 첫 번째 레이어와 나머지 사이에 간격이 생길 수 있습니다. 0.05mm 단계로 신중하게 조정하세요.
:::

## 링잉과 고스팅

링잉 ("고스팅" 또는 "에코잉"이라고도 함)은 날카로운 모서리나 코너 직후 표면에 나타나는 파형 패턴입니다. 패턴이 모서리에서 "메아리"칩니다.

### 원인

- **진동** — 코너에서의 빠른 가속과 감속이 프레임을 통해 진동 전달
- **속도가 너무 높음** — 특히 외벽 속도 100mm/s 이상에서 두드러진 링잉 발생
- **느슨한 부품** — 느슨한 스풀, 진동하는 케이블 슬리브, 불안정하게 설치된 프린터

### 해결책

**Bambu Lab 입력 셰이퍼 (Resonance Compensation):**
```
Bambu Studio → 프린터 → Resonance compensation
Bambu Lab X1C와 P1S는 내장 가속도계를 가지고 자동 보정함
```

**속도 낮추기:**
```
외벽: 60–80 mm/s로 낮추기
가속도: 표준에서 3000–5000 mm/s²로 줄이기
```

**기계적 점검:**
- 프린터가 안정적인 바닥 위에 있는지 확인
- 스풀이 스풀 홀더에서 진동하지 않는지 확인
- 프레임 외부 패널의 접근 가능한 모든 나사를 조이기

:::tip X1C와 P1S는 링잉을 자동 보정
Bambu Lab X1C와 P1S는 시작 시 자동으로 실행되는 내장 가속도계 보정 기능이 있습니다. 일정 기간 후 링잉이 나타나면 유지보수 메뉴에서 "Full calibration"을 실행하세요.
:::

## 언더 압출

언더 압출은 프린터가 플라스틱을 너무 적게 압출할 때 발생합니다. 결과는 얇고 약한 벽, 레이어 간 가시적인 구멍, "거칠고 지저분한" 표면입니다.

### 원인

- **노즐 부분 막힘** — 탄소 축적이 유량 감소
- **노즐 온도가 너무 낮음** — 플라스틱이 충분히 유동적이지 않음
- **압출기 메커니즘의 마모된 기어**가 필라멘트를 충분히 잡지 못함
- **속도가 너무 높음** — 압출기가 원하는 유량을 따라가지 못함

### 해결책

**콜드 풀 (Cold pull):**
```
1. 노즐을 220 °C로 가열
2. 필라멘트를 수동으로 밀어넣기
3. 압력을 유지하면서 노즐을 90 °C (PLA)로 냉각
4. 필라멘트를 빠르게 당겨내기
5. 당겨낸 것이 깨끗할 때까지 반복
```

**온도 조정:**
- 노즐 온도를 5–10 °C 올리고 재테스트
- 너무 낮은 온도로 작동하는 것은 언더 압출의 일반적인 원인

**유량률 보정:**
```
Bambu Studio → 보정 → Flow rate
언더 압출이 사라질 때까지 점진적으로 증가
```

**압출기 기어 확인:**
- 필라멘트를 제거하고 기어 검사
- 기어에 필라멘트 분말이 있으면 작은 브러시로 청소

## 오버 압출

오버 압출은 너무 넓은 가닥을 만들어냅니다 — 표면이 느슨하거나, 광택이 있거나, 불균일하게 보이며 blobs가 생기는 경향이 있습니다.

### 원인

- **유량률 너무 높음** (EM — Extrusion Multiplier)
- **잘못된 필라멘트 직경** — 1.75mm 프로필로 2.85mm 필라멘트를 사용하면 대량 오버 압출
- **노즐 온도 너무 높음**으로 플라스틱이 너무 유동적이 됨

### 해결책

**유량률 보정:**
```
Bambu Studio → 보정 → Flow rate
표면이 균일하고 무광이 될 때까지 2% 단계로 낮추기
```

**필라멘트 직경 확인:**
- 필라멘트를 따라 5–10군데에서 버니어 캘리퍼스로 실제 필라멘트 직경 측정
- 0.05mm 이상의 평균 편차는 저품질 필라멘트를 나타냄

## 필로잉 (Pillowing)

필로잉은 상단 레이어 사이에 "쿠션" 같은 플라스틱이 있는 거품 같고 불균일한 상단 레이어입니다. 낮은 인필과 상단 레이어가 너무 적을 때 특히 두드러집니다.

### 원인

- **상단 레이어 너무 적음** — 인필 위의 플라스틱이 구멍으로 떨어짐
- **냉각 불충분** — 플라스틱이 인필 구멍을 가로질러 다리를 놓기에 충분히 빠르게 굳지 않음
- **인필 너무 낮음** — 인필의 큰 구멍을 덮기 어려움

### 해결책

**상단 레이어 수 늘리기:**
```
Bambu Studio → 품질 → Top shell layers
최소: 4 레이어
균일한 표면 권장: 5–6 레이어
```

**냉각 증가:**
- PLA는 3번째 레이어부터 냉각 팬을 100%로
- 냉각 불충분은 필로잉의 가장 일반적인 원인

**인필 증가:**
- 필로잉이 지속되면 10–15%에서 20–25%로 증가
- Gyroid 패턴은 라인보다 더 균일한 브리징 표면 제공

:::tip 아이어닝 (Ironing)
Bambu Studio의 "ironing" 기능은 표면을 평탄하게 하기 위해 상단 레이어 위를 한 번 더 노즐이 지나갑니다. 최고의 상단 레이어 마감을 위해 품질 → Ironing에서 활성화하세요.
:::

## 레이어 분리 (Layer separation / delamination)

레이어 분리는 레이어들이 서로 제대로 접착되지 않을 때 발생합니다. 최악의 경우 출력물이 레이어 라인을 따라 갈라집니다.

### 원인

- **노즐 온도 너무 낮음** — 플라스틱이 아래 레이어에 충분히 녹아들지 않음
- **냉각 과다** — 플라스틱이 융합되기 전에 너무 빨리 굳음
- **레이어 두께 너무 높음** — 노즐 직경의 80% 이상에서 융합 불량
- **속도 너무 높음** — mm당 경로의 용융 시간 감소

### 해결책

**온도 높이기:**
- 표준에서 +10 °C로 시도하고 관찰
- ABS와 ASA는 특히 민감 — 제어된 챔버 가열 필요

**냉각 줄이기:**
- ABS: 냉각 팬 끄기 (0%)
- PETG: 최대 20–40%
- PLA: 완전 냉각 가능하지만 레이어 분리 발생 시 줄이기

**레이어 두께:**
- 노즐 직경의 최대 75% 사용
- 0.4mm 노즐: 권장 최대 레이어 높이는 0.30mm

**인클로저가 충분히 따뜻한지 확인 (ABS/ASA/PA/PC):**
```
Bambu Lab X1C/P1S: 출력 시작 전 챔버가 40–60 °C로
가열되도록 — 출력 중 문을 열지 마세요
```

## 일반 문제 해결 팁

1. **한 번에 한 가지만 변경** — 각 변경 사이에 작은 보정 출력으로 테스트
2. **먼저 필라멘트 건조** — 많은 표면 결함은 실제로 습기 증상
3. **노즐 청소** — 부분적인 막힘은 진단하기 어려운 일관성 없는 표면 결함을 생성
4. **큰 조정 후 Bambu Studio 유지보수 메뉴에서 전체 보정 실행**
5. **Bambu Dashboard 사용**으로 시간이 지남에 따라 최고의 결과를 낸 설정 추적
