# 필라멘트 건조

습기 찬 필라멘트는 나쁜 출력의 가장 일반적이고 과소평가된 원인 중 하나입니다. 건조해 보이는 필라멘트도 결과를 망칠 만큼 충분한 습기를 흡수했을 수 있습니다 — 특히 나일론과 PVA 같은 소재에서는요.

## 왜 필라멘트를 건조해야 하나요?

많은 플라스틱 유형은 **흡습성**이 있습니다 — 시간이 지남에 따라 공기에서 습기를 흡수합니다. 습기 찬 필라멘트가 뜨거운 노즐을 통과할 때 수분이 갑자기 증발해 녹은 플라스틱에 마이크로 버블을 만듭니다. 결과적으로:

- 출력 중 **팝핑 소리와 딱딱거리는 소리**
- 노즐에서 보이는 **안개나 증기**
- 조정으로 없앨 수 없는 **스트링과 털**
- **거칠거나 알갱이 같은 표면** — 특히 상단 레이어에서
- 불량한 레이어 접착과 마이크로 크랙을 가진 **약한 부품**
- 원래 광택이 있거나 실크 광택이 있어야 하는 소재의 **무광 또는 거품 표면**

:::warning 설정을 조정하기 전에 필라멘트를 건조하세요
많은 사람들이 습기 찬 필라멘트가 원인인데 리트랙과 온도를 몇 시간 동안 조정하고도 개선을 보지 못합니다. 출력 설정을 변경하기 전에 항상 필라멘트를 건조하고 재테스트하세요.
:::

## 어떤 소재가 건조가 필요한가요?

모든 플라스틱 유형이 습해질 수 있지만, 흡습성 정도는 크게 다릅니다:

| 소재 | 흡습성 | 건조 온도 | 건조 시간 | 우선도 |
|---|---|---|---|---|
| PLA | 낮음 | 45–50 °C | 4–6시간 | 선택 |
| PETG | 중간 | 65 °C | 4–6시간 | 권장 |
| ABS | 중간 | 65–70 °C | 4시간 | 권장 |
| TPU | 중간 | 50–55 °C | 4–6시간 | 권장 |
| ASA | 중간 | 65 °C | 4시간 | 권장 |
| PC | 높음 | 70–80 °C | 6–8시간 | 필수 |
| PA/나일론 | 매우 높음 | 70–80 °C | 8–12시간 | 필수 |
| PA-CF | 매우 높음 | 70–80 °C | 8–12시간 | 필수 |
| PVA | 매우 높음 | 45–50 °C | 4–6시간 | 필수 |

:::tip 나일론과 PVA는 중요합니다
PA/나일론과 PVA는 일반 실내 환경에서 **몇 시간 내에** 출력 불가능할 정도의 습기를 흡수할 수 있습니다. 이러한 소재의 새 스풀은 바로 건조하지 않고는 개봉하지 마세요 — 항상 밀폐 박스나 건조 박스에서 출력하세요.
:::

## 습기 찬 필라멘트의 징후

항상 표를 따라 필라멘트를 건조할 필요는 없습니다. 증상을 인식하는 법을 배우세요:

| 증상 | 습기? | 다른 가능한 원인 |
|---|---|---|
| 딱딱거리는/팝핑 소리 | 예, 매우 가능성 높음 | 노즐 부분 막힘 |
| 노즐에서 안개/증기 | 예, 거의 확실 | 다른 원인 없음 |
| 거칠고 알갱이 같은 표면 | 예, 가능 | 온도 너무 낮음, 속도 너무 높음 |
| 사라지지 않는 스트링 | 예, 가능 | 잘못된 리트랙, 온도 너무 높음 |
| 약하고 부서지기 쉬운 부품 | 예, 가능 | 온도 너무 낮음, 잘못된 인필 |
| 색상 변화 또는 무광 표면 | 예, 가능 | 잘못된 온도, 탄 플라스틱 |

## 건조 방법

### 필라멘트 건조기 (권장)

전용 필라멘트 건조기는 가장 간단하고 안전한 솔루션입니다. 정확한 온도를 유지하고 작업 내내 건조기에서 직접 출력할 수 있습니다.

인기 모델:
- **eSun eBOX** — 저렴하고, 박스에서 출력 가능, 대부분의 소재 지원
- **Bambu Lab Filament Dryer** — Bambu AMS에 최적화, 고온 지원
- **Polymaker PolyDryer** — 좋은 온도계와 뛰어난 온도 제어
- **Sunlu S2 / S4** — 예산 친화적, 여러 스풀 동시에

절차:
```
1. 건조기에 스풀 넣기
2. 위 표에서 온도 설정
3. 권장 시간으로 타이머 설정
4. 기다리기 — 과정을 단축하지 마세요
5. 건조기에서 직접 출력하거나 즉시 포장
```

### 식품 건조기

일반 식품 건조기는 필라멘트 건조기로 놀랍도록 잘 작동합니다:

- 구매 가격 저렴 (약 3만원부터)
- 좋은 공기 순환
- 많은 모델에서 70–75 °C까지 온도 지원

:::warning 건조기의 최대 온도 확인
많은 저가 식품 건조기는 온도 조절기가 부정확하여 ±10 °C 변동할 수 있습니다. 높은 열이 필요한 PA와 PC는 온도계로 실제 온도를 측정하세요.
:::

### 오븐

오븐은 긴급 시에 사용할 수 있지만 주의가 필요합니다:

:::danger PLA에 60 °C 이상의 일반 오븐을 절대 사용하지 마세요 — 변형됩니다!
PLA는 55–60 °C에서 이미 부드러워지기 시작합니다. 뜨거운 오븐은 스풀을 손상시키고, 코어를 녹이고, 필라멘트를 사용 불가능하게 만들 수 있습니다. 온도가 정확히 보정되어 50 °C 미만인지 알지 못하는 한 PLA에 오븐을 절대 사용하지 마세요.
:::

더 높은 온도를 견디는 소재(ABS, ASA, PA, PC)의 경우:
```
1. 오븐을 원하는 온도로 예열
2. 온도계로 실제 온도 확인
3. 스풀을 랙 위에 놓기 (오븐 바닥에 직접이 아님)
4. 습기를 빼내기 위해 문을 약간 열어두기
5. 이 방법을 처음 사용할 때 모니터링
```

### Bambu Lab AMS

Bambu Lab AMS Lite와 AMS Pro는 내장 건조 기능(낮은 열 + 공기 순환)이 있습니다. 이것은 완전한 건조의 대체품이 아니지만, 출력 중 이미 건조된 필라멘트를 건조하게 유지합니다.

- AMS Lite: 수동 건조 — 습기 흡수를 제한하지만 적극적으로 건조하지 않음
- AMS Pro: 능동적 가열 — 어느 정도 건조 가능하지만 전용 건조기만큼 효과적이지 않음

## 필라멘트 보관

건조 후 올바른 보관은 건조 과정 자체만큼 중요합니다:

### 최선의 솔루션

1. **실리카겔이 있는 건조 캐비닛** — 습도계와 건조제가 있는 전용 캐비닛. 습도를 안정적으로 낮게 유지 (이상적으로 20% RH 미만)
2. **진공 봉투** — 공기를 빼고 안에 건조제와 함께 밀봉. 가장 저렴한 장기 보관
3. **건조제가 있는 지퍼백** — 짧은 기간에는 간단하고 효과적

### 실리카겔과 건조제

- **파란색/주황색 실리카겔**은 포화 정도를 나타냄 — 색상이 변하면 교체하거나 재생 (120 °C 오븐에서 건조)
- **비드형 실리카겔**은 과립형보다 더 효과적
- 필라멘트 제조업체의 **건조제 팩**은 재생하여 재사용 가능

### 보관함의 습도계

저렴한 디지털 습도계는 박스 내 현재 습도를 표시합니다:

| 상대 습도 (RH) | 상태 |
|---|---|
| 15% 미만 | 이상적 |
| 15–30% | 대부분의 소재에 좋음 |
| 30–50% | PLA와 PETG에는 허용 가능 |
| 50% 초과 | 문제 있음 — 특히 PA, PVA, PC의 경우 |

## 실용적인 팁

- **출력 직전에 건조** — 건조된 필라멘트는 일반 실내 환경에서 며칠 내에 다시 습해질 수 있음
- PA, PC, PVA는 **건조기에서 출력** — 건조 후 그냥 보관하지 마세요
- **새 스풀 ≠ 건조된 스풀** — 제조업체는 건조제로 밀봉하지만 보관 체인에 문제가 있을 수 있음. 흡습성 소재의 새 스풀은 항상 건조하세요
- **건조된 스풀에 건조 날짜 레이블 붙이기**
- 건조기에서 프린터까지의 **전용 PTFE 튜브**로 출력 중 노출 최소화
