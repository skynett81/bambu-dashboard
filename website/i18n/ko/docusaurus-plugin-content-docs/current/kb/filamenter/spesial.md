# 특수 소재

일반적인 소재 외에도 UV 저항 실외 부품부터 수용성 지지 소재까지 특정 용도를 위한 다양한 특수 소재가 있습니다. 다음은 실용적인 개요입니다.

---

## ASA (아크릴로니트릴 스티렌 아크릴레이트)

ASA는 실외 사용을 위한 ABS의 최선의 대안입니다. ABS와 거의 동일하게 출력되지만 햇빛과 날씨를 훨씬 잘 견딥니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 240–260 °C |
| 베드 온도 | 90–110 °C |
| 챔버 온도 | 45–55 °C |
| 부품 냉각 | 0–20% |
| 건조 | 권장 (70 °C / 4–6시간) |

### 특징

- **UV 저항:** 황변이나 균열 없이 장기간 일광 노출을 위해 특별히 설계됨
- **열 안정:** 유리 전이 온도 ~100 °C
- **내충격:** ABS보다 우수한 내충격성
- **인클로저 필요:** ABS와 같은 방식으로 뒤틀림 — X1C/P1S가 최고의 결과

:::tip 실외용 ABS 대신 ASA
부품이 실외(햇빛, 비, 서리)에서 생존해야 하나요? ABS 대신 ASA를 선택하세요. ASA는 눈에 띄는 열화 없이 수년간 지속됩니다. ABS는 수개월 후 균열이 생기고 황변하기 시작합니다.
:::

### 용도
- 실외 브래킷, 하우징, 설치 지점
- 자동차 제상기 부품, 안테나 마운트
- 정원 가구와 실외 환경
- 건물 외부의 표지판과 디스펜서

---

## PC (폴리카보네이트)

폴리카보네이트는 3D 출력 가능한 가장 강하고 내충격성이 높은 플라스틱 중 하나입니다. 투명하고 극단적인 온도를 견딥니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 260–310 °C |
| 베드 온도 | 100–120 °C |
| 챔버 온도 | 50–70 °C |
| 부품 냉각 | 0–20% |
| 건조 | 필수 (80 °C / 8–12시간) |

:::danger PC는 올메탈 핫엔드와 고온이 필요합니다
PC는 표준 PLA 온도에서 녹지 않습니다. 올바른 노즐 설정이 있는 Bambu X1C는 PC를 처리할 수 있습니다. 핫엔드의 PTFE 구성 요소가 사용 온도를 견딜 수 있는지 항상 확인하세요 — 표준 PTFE는 240–250 °C를 연속으로 견딜 수 없습니다.
:::

### 특징

- **매우 내충격:** 저온에서도 파손에 강함
- **투명:** 창문, 렌즈, 광학 부품에 사용 가능
- **열 안정:** 유리 전이 온도 ~147 °C — 일반적인 소재 중 최고
- **흡습성:** 빠르게 습기 흡수 — 항상 철저히 건조
- **뒤틀림:** 강한 수축 — 인클로저와 브림 필요

### 용도
- 안전 바이저와 보호 덮개
- 열에 강한 전기 인클로저
- 렌즈 홀더와 광학 부품
- 로봇 프레임과 드론 동체

---

## PP (폴리프로필렌)

폴리프로필렌은 출력하기 가장 어려운 소재 중 하나이지만, 다른 어떤 플라스틱 소재도 맞출 수 없는 독특한 특성을 제공합니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 220–250 °C |
| 베드 온도 | 80–100 °C |
| 부품 냉각 | 20–50% |
| 건조 | 권장 (70 °C / 6시간) |

### 특징

- **화학 저항:** 강산, 염기, 알코올 및 대부분의 용매에 내성
- **가볍고 유연:** 낮은 밀도, 반복적인 굽힘 견딤 (살아있는 경첩 효과)
- **낮은 접착성:** 자체 및 빌드 플레이트에 접착 불량 — 이것이 도전
- **무독성:** 식품 접촉 안전 (색상과 첨가제에 따라 다름)

:::warning PP는 모든 것에 접착이 어렵습니다
PP는 빌드 플레이트에 접착하지 않는 것으로 유명합니다. Engineering Plate에 PP 테이프(Tesa 테이프나 전용 PP 테이프 등)를 사용하거나 PP 전용으로 배합된 접착제 스틱을 사용하세요. 15–20mm의 브림이 필요합니다.
:::

### 용도
- 실험실 병과 화학 용기
- 식품 보관 부품과 주방 용품
- 살아있는 경첩 (수천 번의 열기/닫기 사이클을 견디는 박스 뚜껑)
- 화학물질에 내성 있는 자동차 부품

---

## PVA (폴리비닐 알코올) — 수용성 지지 소재

PVA는 지지 소재로만 사용되는 특수 소재입니다. 물에 녹아 모델에 깨끗한 표면을 남깁니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 180–220 °C |
| 베드 온도 | 35–60 °C |
| 건조 | 중요 (55 °C / 6–8시간) |

:::danger PVA는 극도로 흡습성이 있습니다
PVA는 다른 어떤 일반 필라멘트보다 빠르게 습기를 흡수합니다. 출력 전에 PVA를 철저히 건조하고, 항상 실리카겔이 있는 밀봉 박스에 보관하세요. 습한 PVA는 노즐에 달라붙어 제거하기 매우 어렵습니다.
:::

### 사용 및 용해

1. PVA를 지지 소재로 사용하여 모델 출력 (멀티 소재 프린터 — AMS 필요)
2. 완성된 출력물을 따뜻한 물 (30–40 °C)에 넣기
3. 30–120분 담가두기, 필요 시 물 교체
4. 깨끗한 물로 헹구고 건조

가능하면 **PVA 전용 압출기를 항상 사용** — 표준 압출기의 PVA 잔류물은 다음 출력을 망칠 수 있습니다.

### 용도
- 수동으로 제거 불가능한 복잡한 지지 구조
- 눈에 띄는 표면 자국 없는 내부 오버행 지지
- 공동과 내부 채널이 있는 모델

## HIPS (고충격 폴리스티렌) — 용매 용해성 지지 소재

HIPS는 ABS와 함께 사용하도록 설계된 또 다른 지지 소재입니다. **리모넨** (감귤류 용매)에 녹습니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 220–240 °C |
| 베드 온도 | 90–110 °C |
| 챔버 온도 | 45–55 °C |
| 건조 | 권장 (65 °C / 4–6시간) |

### ABS와의 사용

HIPS는 ABS와 같은 온도에서 출력되고 잘 접착됩니다. 출력 후 30–60분 D-리모넨에 담가두면 HIPS가 녹습니다.

:::warning 리모넨은 물이 아닙니다
D-리모넨은 오렌지 껍질에서 추출한 용매입니다. 비교적 무해하지만 장갑을 사용하고 환기된 곳에서 작업하세요. 사용한 리모넨을 하수구에 버리지 마세요 — 폐기물 처리 센터에 가져가세요.
:::

### 비교: PVA 대 HIPS

| 특징 | PVA | HIPS |
|------|-----|------|
| 용매 | 물 | D-리모넨 |
| 호환 소재 | PLA 호환 | ABS 호환 |
| 습도 민감성 | 극도로 높음 | 중간 |
| 비용 | 높음 | 중간 |
| 가용성 | 좋음 | 중간 |

## PVB / Fibersmooth — 에탄올 평활화 가능 소재

PVB (폴리비닐 부티랄)는 아세톤으로 ABS를 평활화하는 것과 같은 방식으로 **에탄올(알코올)로 평활화**할 수 있는 독특한 소재이지만 훨씬 더 안전합니다.

### 설정

| 파라미터 | 값 |
|---------|-----|
| 노즐 온도 | 190–210 °C |
| 베드 온도 | 35–55 °C |
| 부품 냉각 | 80–100% |
| 건조 | 권장 (55 °C / 4시간) |

### 에탄올 평활화

1. 표준 PVB 설정으로 모델 출력
2. 브러시로 99% 이소프로필 알코올(IPA) 또는 에탄올 도포
3. 10–15분 건조 — 표면이 균일하게 흘러내림
4. 더 매끄러운 표면을 위해 필요 시 반복
5. 또는: 도포 후 밀폐 용기에 5분간 증기 처리

:::tip 아세톤보다 더 안전
IPA/에탄올은 아세톤보다 훨씬 안전하게 취급할 수 있습니다. 인화점이 높고 증기 독성이 훨씬 낮습니다. 그래도 충분한 환기를 권장합니다.
:::

### 용도
- 매끄러운 표면이 필요한 피규어와 장식
- 발표용 프로토타입
- 도색할 부품 — 매끄러운 표면으로 더 나은 도색

| 소재 | 권장 플레이트 | 접착제 스틱? |
|------|------------|-----------|
| ASA | Engineering Plate / High Temp Plate | 예 |
| PC | High Temp Plate | 예 (필수) |
| PP | Engineering Plate + PP 테이프 | PP 전용 테이프 |
| PVA | Cool Plate / Textured PEI | 아니오 |
| HIPS | Engineering Plate / High Temp Plate | 예 |
| PVB | Cool Plate / Textured PEI | 아니오 |
