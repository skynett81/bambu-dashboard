# 출력 프로필과 설정

출력 프로필은 프린터가 정확히 어떻게 작동하는지 결정하는 설정들의 모음입니다 — 온도와 속도부터 리트랙과 레이어 높이까지. 올바른 프로필은 완벽한 출력과 실패한 출력의 차이입니다.

## 출력 프로필이란 무엇인가요?

Bambu Studio는 세 가지 유형의 프로필을 구분합니다:

- **필라멘트 프로필** — 특정 소재의 온도, 냉각, 리트랙, 건조
- **프로세스 프로필** — 레이어 높이, 속도, 인필, 지지 설정
- **프린터 프로필** — 기계 특화 설정 (Bambu Lab 프린터에 자동 설정)

Bambu Studio는 모든 Bambu Lab 필라멘트와 많은 서드파티 소재에 대한 일반 프로필을 제공합니다. Polyalkemi, eSUN, Fillamentum과 같은 서드파티 공급업체는 자체 필라멘트에 맞게 세밀하게 조정된 최적화된 프로필도 만들어냅니다.

프로필은 사용자 간에 자유롭게 가져오기, 내보내기, 공유할 수 있습니다.

## Bambu Studio에서 프로필 가져오기

1. 공급업체 웹사이트 또는 MakerWorld에서 프로필(JSON 파일) 다운로드
2. Bambu Studio 열기
3. **파일 → 가져오기 → 구성 가져오기**로 이동
4. 다운로드한 파일 선택
5. 슬라이서의 필라멘트 선택에 프로필이 나타남

:::tip 정리
예를 들어 "Polyalkemi PLA HF 0.20mm Balanced"와 같은 설명적인 이름을 프로필에 부여하여 다음 번에 올바른 프로필을 쉽게 찾을 수 있도록 하세요.
:::

## 중요한 설정 설명

### 온도

온도는 가장 중요한 단일 설정입니다. 온도가 너무 낮으면 레이어 접착 불량과 부족한 채움이 발생합니다. 너무 높으면 스트링, 기포 표면, 탄 필라멘트가 발생합니다.

| 설정 | 설명 | 일반적인 PLA | 일반적인 PETG | 일반적인 ABS |
|---|---|---|---|---|
| 노즐 온도 | 용융 온도 | 200–220 °C | 230–250 °C | 240–260 °C |
| 베드 온도 | 빌드 플레이트 가열 | 55–65 °C | 70–80 °C | 90–110 °C |
| 챔버 온도 | 인클로저 온도 | 불필요 | 선택 | 40–60 °C 권장 |

Bambu Lab X1C와 P1 시리즈는 능동적 챔버 가열 기능이 있습니다. ABS와 ASA의 경우 뒤틀림과 레이어 분리를 방지하기 위해 이것이 중요합니다.

### 속도

Bambu Lab 프린터는 매우 빠르게 실행될 수 있지만, 더 높은 속도가 항상 더 나은 결과를 의미하지는 않습니다. 특히 외벽 속도가 표면에 영향을 줍니다.

| 설정 | 영향 | 품질 모드 | 균형 | 고속 |
|---|---|---|---|---|
| 외벽 | 표면 결과 | 45–60 mm/s | 100 mm/s | 150+ mm/s |
| 내벽 | 구조적 강도 | 100 mm/s | 150 mm/s | 200+ mm/s |
| 인필 | 내부 채움 | 150 mm/s | 200 mm/s | 300+ mm/s |
| 상단 레이어 | 상단 표면 | 45–60 mm/s | 80 mm/s | 100 mm/s |
| 하단 레이어 | 첫 번째 레이어 | 30–50 mm/s | 50 mm/s | 50 mm/s |

:::tip 외벽 속도가 표면 품질의 열쇠
실크 광택 마감을 위해 외벽 속도를 45–60 mm/s로 낮추세요. 이것은 특히 Silk PLA와 매트 필라멘트에 적용됩니다. 내벽과 인필은 표면에 영향 없이 여전히 빠르게 실행될 수 있습니다.
:::

### 리트랙 (되감기)

리트랙은 프린터가 압출 없이 이동할 때 필라멘트를 노즐에 약간 되감습니다. 이것은 스트링(부품 사이의 가는 실)을 방지합니다. 잘못된 리트랙 설정은 스트링(너무 적음) 또는 잼(너무 많음)을 만들어냅니다.

| 소재 | 리트랙 거리 | 리트랙 속도 | 비고 |
|---|---|---|---|
| PLA | 0.8–2.0 mm | 30–50 mm/s | 대부분 표준 |
| PETG | 1.0–3.0 mm | 20–40 mm/s | 너무 많음 = 잼 |
| ABS | 0.5–1.5 mm | 30–50 mm/s | PLA와 유사 |
| TPU | 0–1.0 mm | 10–20 mm/s | 최소화! 또는 비활성화 |
| 나일론 | 1.0–2.0 mm | 30–40 mm/s | 건조한 필라멘트 필요 |

:::warning TPU 리트랙
TPU와 기타 유연한 소재: 최소 리트랙(0–1 mm)을 사용하거나 완전히 비활성화하세요. 리트랙이 너무 많으면 부드러운 필라멘트가 휘어 Bowden 튜브에서 막혀 잼을 일으킵니다.
:::

### 레이어 높이

레이어 높이는 세부 수준과 출력 속도 사이의 균형을 결정합니다. 낮은 레이어 높이는 더 세밀한 디테일과 더 부드러운 표면을 만들어내지만, 훨씬 더 많은 시간이 필요합니다.

| 레이어 높이 | 설명 | 용도 |
|---|---|---|
| 0.08 mm | 초고정밀 | 미니어처, 세밀한 모델 |
| 0.12 mm | 세밀함 | 시각적 품질, 텍스트, 로고 |
| 0.16 mm | 고품질 | 대부분 출력의 표준 |
| 0.20 mm | 균형 | 시간/품질의 좋은 균형 |
| 0.28 mm | 고속 | 기능성 부품, 프로토타입 |

Bambu Studio는 "0.16mm High Quality"와 "0.20mm Balanced Quality" 같은 프로세스 설정으로 작동하며 — 이것들은 레이어 높이를 설정하고 속도와 냉각을 자동으로 조정합니다.

### 인필 (채움)

인필은 출력물 내부를 채우는 소재의 양을 결정합니다. 더 많은 인필 = 더 강하고, 무겁고, 긴 출력 시간.

| 비율 | 용도 | 권장 패턴 |
|---|---|---|
| 10–15% | 장식, 시각적 | Gyroid |
| 20–30% | 일반 사용 | Cubic, Gyroid |
| 40–60% | 기능성 부품 | Cubic, Honeycomb |
| 80–100% | 최대 강도 | Rectilinear |

:::tip Gyroid는 왕
Gyroid 패턴은 최고의 강도 대 중량비를 제공하고 등방성입니다 — 모든 방향에서 동일하게 강합니다. Honeycomb보다 빠르게 출력되고 열린 모델에서 보기 좋습니다. 대부분의 상황에서 표준 선택입니다.
:::

## 소재별 프로필 팁

### PLA — 품질 집중

PLA는 관대하고 작업하기 쉽습니다. 표면 품질에 집중:

- **외벽:** Silk PLA 특히, 완벽한 표면을 위해 60 mm/s
- **냉각 팬:** 3번째 레이어부터 100% — 날카로운 디테일과 브리지에 중요
- **브림:** 올바르게 보정된 플레이트의 순수 PLA에는 불필요
- **레이어 높이:** 0.16mm High Quality는 장식 부품에 좋은 균형 제공

### PETG — 균형

PETG는 PLA보다 강하지만 세밀한 조정이 더 어렵습니다:

- **프로세스 설정:** 0.16mm High Quality 또는 0.20mm Balanced Quality
- **냉각 팬:** 30–50% — 냉각이 너무 많으면 레이어 접착 불량과 부서지기 쉬운 출력물 발생
- **Z-hop:** 이동 시 노즐이 표면을 끌지 않도록 활성화
- **스트링:** 온도를 낮추는 것보다 약간 높이고 리트랙 조정

### ABS — 인클로저가 전부

ABS는 잘 출력되지만 제어된 환경이 필요합니다:

- **냉각 팬:** 끄기 (0%) — 완전히 중요! 냉각은 레이어 분리와 뒤틀림을 일으킴
- **인클로저:** 문을 닫고 출력 시작 전 챔버가 40–60 °C로 가열되게 함
- **브림:** 크고 평평한 부품에는 5–8mm 권장 — 코너 뒤틀림 방지
- **환기:** 방에 충분한 환기 확보 — ABS는 스티렌 증기를 방출함

### TPU — 느리고 조심스럽게

유연한 소재는 완전히 다른 접근이 필요합니다:

- **속도:** 최대 30 mm/s — 너무 빠른 출력은 필라멘트를 휘게 함
- **리트랙:** 최소 (0–1 mm) 또는 완전히 비활성화
- **다이렉트 드라이브:** TPU는 내장 다이렉트 드라이브가 있는 Bambu Lab 기계에서만 작동
- **레이어 높이:** 0.20mm Balanced는 과도한 장력 없이 좋은 레이어 융합 제공

### 나일론 — 건조한 필라멘트가 전부

나일론은 흡습성이 있고 수 시간 내에 습기를 흡수합니다:

- **항상 건조:** 출력 전 70–80 °C에서 8–12시간
- **인클로저:** 필라멘트를 건조하게 유지하기 위해 건조 박스에서 AMS로 직접 실행
- **리트랙:** 적당히 (1.0–2.0 mm) — 습한 나일론은 훨씬 더 많은 스트링 발생
- **빌드 플레이트:** 접착제가 있는 Engineering Plate, 또는 최고의 접착을 위해 Garolite 플레이트

## Bambu Lab 사전 설정

Bambu Studio는 Bambu Lab 전체 제품군을 위한 내장 프로필이 있습니다:

- Bambu Lab Basic PLA, PETG, ABS, TPU, PVA, PA, PC, ASA
- Bambu Lab 지지 소재 (Support W, Support G)
- Bambu Lab Specialty (PLA-CF, PETG-CF, ABS-GF, PA-CF, PPA-CF, PPA-GF)
- 일반 프로필 (Generic PLA, Generic PETG 등) — 서드파티 필라멘트의 시작점으로 기능

일반 프로필은 좋은 시작점입니다. 실제 필라멘트를 기반으로 ±5 °C로 온도를 미세 조정하세요.

## 서드파티 프로필

많은 선도 공급업체가 자체 필라멘트에 맞게 최적화된 바로 사용 가능한 Bambu Studio 프로필을 제공합니다:

| 공급업체 | 사용 가능한 프로필 | 다운로드 |
|---|---|---|
| [Polyalkemi](https://polyalkemi.no) | PLA, PLA High Speed, PETG, PETG-CF, ABS | [Bambu Lab 프로필](https://gammel.polyalkemi.no/bambulabprofiler/) |
| [eSUN](https://www.esun3d.com) | PLA+, ePLA-Lite, PETG, eABS | [eSUN 프로필](https://www.esun3d.com/bambu-lab-3d-printer-filament-setting/) |
| [Fillamentum](https://fillamentum.com) | Nonoilen PLA, PLA, PET-G | [Fillamentum 프로필](https://fillamentum.com/pages/bambu-lab-print-profiles) |
| [Spectrum](https://spectrumfilaments.com) | PETG FR V0, PETG-HT100 | [Spectrum 프로필](https://spectrumfilaments.com/bambu-lab-profiles/) |
| [Fiberlogy](https://fiberlogy.com) | Easy-PETG, Matte-PETG, TPU 30D | [Fiberlogy 프로필](https://fiberlogy.com/en/printing-profiles/) |
| [add:north](https://addnorth.com) | PLA, PETG, AduraX, X-PLA | [add:north 프로필](https://addnorth.com/printing-profiles) |

:::info 프로필을 어디서 찾나요?
- **Bambu Studio:** Bambu Lab 소재와 많은 서드파티를 위한 내장 프로필
- **공급업체 웹사이트:** 다운로드 섹션에서 "Bambu Studio profile" 또는 "JSON profile" 검색
- **Bambu Dashboard:** 도구 섹션의 출력 프로필 패널
- **MakerWorld:** 프로필은 종종 다른 사용자들의 모델과 함께 공유됨
:::

## 프로필 내보내기 및 공유

개인 프로필을 내보내고 다른 사람들과 공유할 수 있습니다:

1. **파일 → 내보내기 → 구성 내보내기**로 이동
2. 내보낼 프로필(필라멘트, 프로세스, 프린터) 선택
3. JSON 파일로 저장
4. 파일을 직접 공유하거나 MakerWorld에 업로드

Bambu Studio 재설치 시 보존하고 싶은 프로필을 오랜 기간 미세 조정해 왔다면 특히 유용합니다.

## 프로필로 문제 해결

### 스트링

출력된 부품 사이의 가는 실 — 이 순서로 시도:

1. 리트랙 거리를 0.5mm 증가
2. 출력 온도를 5 °C 낮추기
3. "Wipe on retract" 활성화
4. 필라멘트가 건조한지 확인

### 부족한 채움 / 벽의 구멍

출력물이 단단해 보이지 않거나 구멍이 있음:

1. 필라멘트 직경 설정이 올바른지 확인 (1.75 mm)
2. Bambu Studio에서 유량률 보정 (보정 → Flow Rate)
3. 온도를 5 °C 높이기
4. 노즐 부분 막힘 확인

### 레이어 접착 불량

레이어들이 잘 결합되지 않음:

1. 온도를 5–10 °C 높이기
2. 냉각 팬 줄이기 (특히 PETG와 ABS)
3. 출력 속도 낮추기
4. 인클로저가 충분히 따뜻한지 확인 (ABS/ASA의 경우)
