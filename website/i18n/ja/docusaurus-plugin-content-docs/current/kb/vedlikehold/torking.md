# フィラメントの乾燥

湿ったフィラメントは、悪いプリントの最も一般的で過小評価されている原因の一つです。乾いているように見えるフィラメントでも、結果を台無しにするのに十分な湿気を吸収している可能性があります — 特にナイロンやPVAのような素材では。

## なぜフィラメントを乾燥させるのか？

多くのプラスチック素材は**吸湿性**があります — 時間の経過とともに空気から湿気を吸収します。湿ったフィラメントが熱いノズルを通過すると、水分が突然蒸発して溶けたプラスチックにマイクロバブルを生じさせます。結果として：

- プリント中の**パチパチという音とクラッキング音**
- ノズルから見える**霧や蒸気**
- 調整で取り除けない**ストリングとヘアリング**
- **ざらざらまたは粒状の表面** — 特にトップ層に
- マイクロクラックと層接着不良による**弱い部品**
- 通常は光沢があるかシルク光沢があるはずの素材の**マットまたは泡立った仕上がり**

:::warning 設定を調整する前にフィラメントを乾燥させる
多くの人がリトラクトと温度を何時間も調整しても改善が見られません — 原因が湿ったフィラメントだからです。プリント設定を変更する前に必ずフィラメントを乾燥させて再テストしてください。
:::

## どの素材に乾燥が必要か？

すべてのプラスチック素材は湿ることがありますが、吸湿性の度合いは大きく異なります：

| 素材 | 吸湿性 | 乾燥温度 | 乾燥時間 | 優先度 |
|---|---|---|---|---|
| PLA | 低 | 45–50 °C | 4–6時間 | 任意 |
| PETG | 中 | 65 °C | 4–6時間 | 推奨 |
| ABS | 中 | 65–70 °C | 4時間 | 推奨 |
| TPU | 中 | 50–55 °C | 4–6時間 | 推奨 |
| ASA | 中 | 65 °C | 4時間 | 推奨 |
| PC | 高 | 70–80 °C | 6–8時間 | 必須 |
| PA/ナイロン | 非常に高 | 70–80 °C | 8–12時間 | 必須 |
| PA-CF | 非常に高 | 70–80 °C | 8–12時間 | 必須 |
| PVA | 非常に高 | 45–50 °C | 4–6時間 | 必須 |

:::tip ナイロンとPVAは重要
PA/ナイロンとPVAは通常の屋内環境で**数時間**以内に印刷不可能になるほどの湿気を吸収できます。これらの素材の新しいスプールは乾燥させずに開封しないでください — そして常に閉じたボックスまたは乾燥ボックスから印刷してください。
:::

## 湿ったフィラメントのサイン

テーブルに従って常にフィラメントを乾燥させる必要はありません。症状を認識することを学んでください：

| 症状 | 湿気？ | 他の可能性のある原因 |
|---|---|---|
| クラッキング/パチパチ音 | はい、非常に可能性が高い | ノズルの部分的な詰まり |
| ノズルからの霧/蒸気 | はい、ほぼ確実 | 他の原因はない |
| ざらざら、粒状の表面 | はい、可能性あり | 温度が低すぎる、速度が高すぎる |
| 消えないストリング | はい、可能性あり | リトラクト設定ミス、温度が高すぎる |
| 弱くて脆い部品 | はい、可能性あり | 温度が低すぎる、インフィル設定ミス |
| 色の変化またはマット仕上げ | はい、可能性あり | 温度ミス、焦げたプラスチック |

## 乾燥方法

### フィラメント乾燥機（推奨）

専用フィラメント乾燥機は最もシンプルで安全な解決策です。正確な温度を維持し、作業全体を通じて乾燥機から直接印刷できます。

人気モデル：
- **eSun eBOX** — リーズナブル、ボックスから印刷可能、ほとんどの素材に対応
- **Bambu Lab Filament Dryer** — Bambu AMS向けに最適化、高温対応
- **Polymaker PolyDryer** — 良い温度計と優れた温度制御
- **Sunlu S2 / S4** — 予算に優しく、複数スプールを同時に

手順：
```
1. 乾燥機にスプールを入れる
2. 上の表から温度を設定する
3. 推奨時間にタイマーをセットする
4. 待つ — プロセスを短縮しない
5. 乾燥機から直接印刷するか、すぐに包む
```

### 食品乾燥機

一般的な食品乾燥機はフィラメント乾燥機として驚くほどうまく機能します：

- 購入がリーズナブル（約3,000円から）
- 良好な空気循環
- 多くのモデルで70–75 °Cまでの温度に対応

:::warning 乾燥機の最大温度を確認する
多くの安価な食品乾燥機はサーモスタットが不正確で±10 °C変動することがあります。高温が必要なPAとPCは温度計で実際の温度を測定してください。
:::

### オーブン

オーブンは緊急時に使用できますが注意が必要です：

:::danger PLA に絶対に60 °C以上の通常オーブンを使用しないでください — 変形します！
PLAは55–60 °Cですでに軟化し始めます。熱いオーブンはスプールを損傷し、コアを溶かし、フィラメントを使用不能にする可能性があります。温度が正確に較正されて50 °C未満であることがわかっている場合を除いて、PLAにオーブンを絶対に使用しないでください。
:::

高温に耐える素材（ABS、ASA、PA、PC）の場合：
```
1. 希望温度にオーブンを予熱する
2. 温度計で実際の温度を確認する
3. ラックにスプールを置く（オーブンの底に直接ではなく）
4. 湿気を逃がすためにドアを少し開けておく
5. この方法を初めて使用するときは監視する
```

### Bambu Lab AMS

Bambu Lab AMS Lite と AMS Pro には内蔵乾燥機能（低温 + 空気循環）があります。これは完全な乾燥の代替品ではありませんが、プリント中に既に乾燥したフィラメントを乾燥した状態に保ちます。

- AMS Lite：受動的乾燥 — 湿気の吸収を制限するが、積極的に乾燥しない
- AMS Pro：積極的な加熱 — ある程度の乾燥が可能だが、専用乾燥機ほど効果的ではない

## フィラメントの保管

乾燥後の適切な保管は乾燥プロセス自体と同じくらい重要です：

### 最良の解決策

1. **シリカゲル入り乾燥キャビネット** — 湿度計と乾燥剤を備えた専用キャビネット。湿度を安定して低く保つ（理想的には20% RH未満）
2. **真空パック袋** — 空気を吸い出して内側に乾燥剤で密封する。最も安い長期保管方法
3. **乾燥剤入りジップロック袋** — 短期間には簡単で効果的

### シリカゲルと乾燥剤

- **青/オレンジのシリカゲル**は飽和度を示す — 色が変わったら交換または再生（120 °Cのオーブンで乾燥）
- **ビーズ状シリカゲル**は顆粒よりも効果的
- フィラメントメーカーからの**乾燥剤パック**は再生して再利用できる

### 保管ボックスの湿度計

安価なデジタル湿度計はボックス内の現在の湿度を表示します：

| 相対湿度（RH） | 状態 |
|---|---|
| 15%未満 | 理想的 |
| 15–30% | ほとんどの素材に良好 |
| 30–50% | PLA と PETG には許容範囲 |
| 50%以上 | 問題あり — 特に PA、PVA、PC の場合 |

## 実践的なヒント

- **印刷直前に乾燥させる** — 乾燥したフィラメントは通常の屋内環境で数日で再び湿ることがある
- PA、PC、PVA は**乾燥機から印刷する** — 乾燥してしまってから放置しない
- **新しいスプール ≠ 乾燥したスプール** — メーカーは乾燥剤で密封するが、保管チェーンが問題を抱えている可能性がある。吸湿性の素材の新しいスプールは常に乾燥させる
- **乾燥したスプールに乾燥日付のラベルを貼る**
- 乾燥機からプリンターへの**専用PTFEチューブ**で印刷中の露出を最小化する

## Bambu Dashboard と乾燥状態

Bambu Dashboard では、フィラメントプロファイルの下で最後の乾燥日を含むフィラメント情報をログに記録できます。これを使って、どのスプールが最近乾燥したか、どれが新しいサイクルが必要かを追跡してください。
