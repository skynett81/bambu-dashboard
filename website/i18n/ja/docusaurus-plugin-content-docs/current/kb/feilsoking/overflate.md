# 表面欠陥

3Dプリントの表面は、システム内部で何が起きているかを多く語っています。ほとんどの表面欠陥には1〜2つの明確な原因があり、正しく診断すれば驚くほど簡単に修正できます。

## クイック診断概要

| 症状 | 最も一般的な原因 | 最初の対処 |
|---|---|---|
| Blobs と zits | 過剰押出、シーム配置 | シームを調整、フロー較正 |
| 可視レイヤーライン | Z-wobble、層が厚すぎる | より細かい層に変更、Z軸を確認 |
| Elephant foot | 最初の層が広すぎる | Elephant foot compensation |
| リンギング/ゴースティング | 高速時の振動 | 速度を下げ、input shaperを有効化 |
| アンダー押出 | ノズル詰まり、温度が低すぎる | ノズルを清掃、温度を上げる |
| オーバー押出 | フローレートが高すぎる | フローレートを較正 |
| ピロウイング | トップ層が少なすぎる、冷却不足 | トップ層を増やす、冷却ファンを増やす |
| 層分離 | 温度が低すぎる、冷却しすぎ | 温度を上げる、冷却ファンを減らす |

---

## Blobs と zits

Blobsは表面の不規則な塊です。Zitsは点のような点で、通常シームラインに沿って現れます。

### 原因

- **過剰押出** — 過剰な樹脂が押し出され横に溢れる
- **シーム配置が悪い** — 標準の「nearest」シームは全ての移行点を同じ場所に集める
- **リトラクト問題** — 不十分なリトラクトがノズルに圧力を蓄積させる
- **湿ったフィラメント** — 湿気がマイクロバブルと滴りを生じさせる

### 解決策

**Bambu Studio でのシーム設定：**
```
Bambu Studio → 品質 → Seam position
- Aligned: 全シームを同じ場所に（見えるが整然とした）
- Nearest: 最近点（blobsをランダムに分散）
- Back: オブジェクトの裏側（視覚的品質に推奨）
- Random: ランダム分散（シームを最も効果的に隠す）
```

**フローレート較正：**
```
Bambu Studio → 較正 → Flow rate
±2% ステップで調整してblobsが消えるまで
```

:::tip 視覚的品質のために「Back」にシームを設定
オブジェクトの裏側にシームを配置して最も見えにくくします。より清潔なシーム仕上げのために「Wipe on retract」と組み合わせてください。
:::

## 可視レイヤーライン

すべてのFDMプリントにはレイヤーラインがありますが、通常のプリントでは一貫していてほとんど見えないはずです。異常な可視性は具体的な問題を示しています。

### 原因

- **Z-wobble** — Z軸が振動または曲がっており、全高にわたる波状パターンを生じさせる
- **層が厚すぎる** — 0.28mm以上のレイヤー高さは完璧なプリントでも目立つ
- **温度変動** — 不安定な溶融温度が層幅を変動させる
- **不均一なフィラメント径** — 直径が変動する安価なフィラメント

### 解決策

**Z-wobble：**
- スクリュースピンドル（Z-leadscrew）が清潔で潤滑されていることを確認
- スピンドルが曲がっていないことを確認（回転時に目視検査）
- [Z軸の潤滑](/docs/kb/vedlikehold/smoring)のメンテナンス記事を参照

**レイヤー高さ：**
- より均一な表面のために0.12mmまたは0.16mmに変更
- レイヤー高さを半分にするとプリント時間が倍になることを覚えておいてください

**温度変動：**
- PID較正を使用（Bambu Studioのメンテナンスメニューで利用可能）
- プリント中にノズルを冷却する空気の流れを避ける

## Elephant foot

Elephant footは最初の層がオブジェクトの残りより広い場合に発生します — オブジェクトが底で「広がる」ようなイメージです。

### 原因

- 最初の層がプレートに対して強く押しつぶされる（Z-offsetが密すぎる）
- ベッド温度が高すぎて樹脂が長時間柔らかく流体状態を維持する
- 最初の層の冷却が不十分で樹脂が広がる時間が多すぎる

### 解決策

**Bambu Studio での Elephant foot compensation：**
```
Bambu Studio → 品質 → Elephant foot compensation
0.1–0.2mm から始めて足が消えるまで調整
```

**Z-offset：**
- 最初のレイヤー高さを再較正する
- 足がなくなるまで0.05mmずつZ-offsetを上げる

**ベッド温度：**
- ベッド温度を5–10 °C下げる
- PLAの場合：55 °Cで十分なことが多い — 65 °CではElephant footになることがある

:::warning 過補正しないでください
Elephant foot compensationを高くしすぎると最初の層と残りの間に隙間ができることがあります。0.05mmのステップで慎重に調整してください。
:::

## リンギングとゴースティング

リンギング（「ゴースティング」または「エコーイング」とも呼ばれる）は、鋭いエッジやコーナーの直後の表面に現れる波状パターンです。パターンがエッジから「反響」します。

### 原因

- **振動** — コーナーでの急速な加速と減速がフレームを通じて振動を伝える
- **速度が高すぎる** — 特に100mm/s以上の外壁速度では顕著なリンギングが発生
- **緩んだ部品** — 緩んだスプール、振動するケーブルホルダー、不安定に設置されたプリンター

### 解決策

**Bambu Lab input shaper（Resonance Compensation）：**
```
Bambu Studio → プリンター → Resonance compensation
Bambu Lab X1C と P1S は内蔵加速度計を持ち自動較正する
```

**速度を下げる：**
```
外壁：60–80 mm/s に下げる
加速度：標準から3000–5000 mm/s² に減らす
```

**機械的チェック：**
- プリンターが安定した面の上にあることを確認
- スプールがスプールホルダーで振動していないことを確認
- フレームの外部パネルのアクセス可能なすべてのネジを締める

:::tip X1C と P1S はリンギングを自動較正する
Bambu Lab X1C と P1S は起動時に自動的に実行される内蔵加速度計較正を持っています。リンギングが一定期間後に現れた場合は、メンテナンスメニューから「Full calibration」を実行してください。
:::

## アンダー押出

アンダー押出はプリンターが樹脂を少なく押し出す場合に発生します。結果は薄くて弱い壁、層間の可視穴、「毛羽立った」表面です。

### 原因

- **ノズルの部分的な詰まり** — カーボン蓄積が流量を減少させる
- **ノズル温度が低すぎる** — 樹脂が十分に流体でない
- **摩耗したギア** — エクストルーダー機構がフィラメントを十分に把持しない
- **速度が高すぎる** — エクストルーダーが望ましいフローに追いつけない

### 解決策

**コールドプル（Cold pull）：**
```
1. ノズルを220 °C に加熱
2. フィラメントを手動で押し込む
3. 圧力を保ちながらノズルを90 °C（PLA）まで冷却
4. フィラメントを素早く引き出す
5. 引き出されたものが清潔になるまで繰り返す
```

**温度調整：**
- ノズル温度を5–10 °C上げて再テスト
- 低すぎる温度での動作はアンダー押出の一般的な原因

**フローレート較正：**
```
Bambu Studio → 較正 → Flow rate
アンダー押出が消えるまで徐々に増やす
```

**エクストルーダーギアを確認：**
- フィラメントを取り外してギアを検査
- 歯にフィラメント粉末がある場合は小さなブラシで清掃

## オーバー押出

オーバー押出は広すぎるコードを生み出します — 表面は緩んでいる、光沢がある、または不均一に見え、blobsが生じやすいです。

### 原因

- **フローレートが高すぎる**（EM — Extrusion Multiplier）
- **誤ったフィラメント径** — 1.75mmプロファイルで2.85mmフィラメントを使うと大規模なオーバー押出
- **ノズル温度が高すぎる**と樹脂が流体になりすぎる

### 解決策

**フローレート較正：**
```
Bambu Studio → 較正 → Flow rate
表面が均一で艶消しになるまで2%ステップで下げる
```

**フィラメント径を確認：**
- フィラメントに沿った5–10箇所でノギスを使って実際のフィラメント径を測定
- 0.05mm以上の平均偏差は低品質フィラメントを示す

## ピロウイング

ピロウイングはトップ層間に「クッション」状の樹脂を持つ泡立ったような不均一なトップ層です。低インフィルと少なすぎるトップ層で特に顕著です。

### 原因

- **トップ層が少なすぎる** — インフィル上の樹脂が穴に落ちる
- **冷却が不十分** — 樹脂がインフィルの穴を橋渡しするのに十分速く固まらない
- **インフィルが低すぎる** — インフィルの大きな穴は橋渡しが難しい

### 解決策

**トップ層数を増やす：**
```
Bambu Studio → 品質 → Top shell layers
最小：4層
均一な表面には推奨：5–6層
```

**冷却を増やす：**
- PLAは3層目から冷却ファンを100%にする
- 冷却不足はピロウイングの最も一般的な原因

**インフィルを増やす：**
- ピロウイングが続く場合は10–15%から20–25%に上げる
- Gyroidパターンはラインより均一な橋渡し面を提供する

:::tip アイロニング
Bambu Studioの「ironing」機能はノズルをトップ層の上で追加で走らせて表面を平滑にします。最高のトップ層仕上げのために品質 → Ironing で有効化してください。
:::

## 層分離（Layer separation / delamination）

層分離は層が適切に互いに接着しない場合に発生します。最悪の場合、プリントがレイヤーラインに沿って割れます。

### 原因

- **ノズル温度が低すぎる** — 樹脂が下の層に十分に溶け込まない
- **冷却しすぎる** — 樹脂が融合する前に早く固まりすぎる
- **層厚が高すぎる** — ノズル径の80%以上では融合が悪くなる
- **速度が高すぎる** — mmあたりの溶融時間が減少する

### 解決策

**温度を上げる：**
- 標準から+10 °Cで試して観察する
- ABS と ASA は特に敏感 — 制御されたチャンバー加熱が必要

**冷却を減らす：**
- ABS：冷却ファンをオフ（0%）
- PETG：最大20–40%
- PLA：完全冷却に耐えられるが、層分離が発生したら減らす

**層厚：**
- ノズル径の最大75%を使用
- 0.4mmノズルの場合：推奨最大レイヤー高さは0.30mm

**エンクロージャーが十分温かいことを確認（ABS/ASA/PA/PC）：**
```
Bambu Lab X1C/P1S：プリント開始前にチャンバーを40–60 °C に
加熱させる — プリント中はドアを開けない
```

## 一般的なトラブルシューティングのヒント

1. **一度に一つだけ変更する** — 各変更の間に小さな較正プリントでテストする
2. **まずフィラメントを乾燥させる** — 多くの表面欠陥は実際には湿気の症状
3. **ノズルを清掃する** — 部分的な詰まりは診断が難しい不安定な表面欠陥を生じさせる
4. **大きな調整後はBambu Studioのメンテナンスメニューから完全較正を実行する**
5. **Bambu Dashboard を使用して**時間の経過とともに最良の結果をもたらした設定を追跡する
