# 特殊素材

一般的な素材に加えて、UV耐性の屋外部品から水溶性サポート素材まで、特定の用途向けの多数の特殊素材があります。実用的な概要を以下に示します。

---

## ASA（アクリロニトリルスチレンアクリレート）

ASAは屋外使用においてABSの最良の代替品です。ABSとほぼ同じように印刷されますが、日光と天候をはるかによく耐えます。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 240–260 °C |
| ベッド温度 | 90–110 °C |
| チャンバー温度 | 45–55 °C |
| 部品冷却 | 0–20% |
| 乾燥 | 推奨（70 °C / 4–6時間） |

### 特徴

- **UV耐性：** 黄ばみや割れなく長期間の日光露出専用に設計
- **熱安定：** ガラス転移温度 ~100 °C
- **耐衝撃：** ABSより優れた耐衝撃性
- **エンクロージャー必要：** ABSと同様に反る — X1C/P1Sが最良の結果

:::tip 屋外用ABSの代わりにASA
部品は北欧の気候（日光、雨、霜）で屋外に置かれますか？ABSの代わりにASAを選んでください。ASAは目に見える劣化なく何年も持ちます。ABSは数ヶ月で割れ始め黄ばみます。
:::

### 用途
- 屋外ブラケット、ハウジング、取り付けポイント
- カーデフロスター部品、アンテナマウント
- ガーデン家具と屋外環境
- 建物外側の看板とディスペンサー

---

## PC（ポリカーボネート）

ポリカーボネートは3Dプリントできる最も強くて耐衝撃性の高いプラスチックの一つです。透明で極端な温度に耐えます。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 260–310 °C |
| ベッド温度 | 100–120 °C |
| チャンバー温度 | 50–70 °C |
| 部品冷却 | 0–20% |
| 乾燥 | 必須（80 °C / 8–12時間） |

:::danger PC はオールメタルホットエンドと高温が必要
PCは標準PLA温度では溶融しません。Bambu X1Cは適切なノズル設定でPCを処理できます。ホットエンドのPTFEコンポーネントが温度に耐えられることを常に確認してください — 標準PTFEは240–250 °C連続に耐えられません。
:::

### 特徴

- **非常に耐衝撃：** 低温でも破壊に強い
- **透明：** 窓、レンズ、光学部品に使用可能
- **熱安定：** ガラス転移温度 ~147 °C — 一般的な素材の中で最高
- **吸湿性：** 急速に湿気を吸収 — 常に徹底的に乾燥させる
- **反り：** 強い収縮 — エンクロージャーとブリムが必要

### 用途
- セーフティーバイザーと保護カバー
- 熱に耐える電気エンクロージャー
- レンズホルダーと光学部品
- ロボットフレームとドローン胴体

---

## PP（ポリプロピレン）

ポリプロピレンは印刷が最も難しい素材の一つですが、他のプラスチック素材では匹敵できないユニークな特性を提供します。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 220–250 °C |
| ベッド温度 | 80–100 °C |
| 部品冷却 | 20–50% |
| 乾燥 | 推奨（70 °C / 6時間） |

### 特徴

- **耐化学薬品：** 強酸、塩基、アルコール、ほとんどの溶剤に耐える
- **軽量で柔軟：** 低密度、繰り返しの曲げに耐える（リビングヒンジ効果）
- **低接着性：** 自身および造形プレートに接着しにくい — これが課題
- **無毒：** 食品接触に安全（色と添加剤に依存）

:::warning PPはすべてに接着しにくい
PPは造形プレートに接着しないことで知られています。Engineering PlateにPPテープ（Tesaテープまたは専用PPテープなど）を使用するか、PP専用に配合されたのりスティックを使用してください。15–20mmのブリムが必要です。
:::

### 用途
- 実験室用ボトルと化学容器
- 食品保管部品とキッチン用品
- リビングヒンジ（何千回もの開閉サイクルに耐えるボックス蓋）
- 化学薬品に耐える自動車部品

---

## PVA（ポリビニルアルコール）— 水溶性サポート素材

PVAはサポート素材としてのみ使用される特殊素材です。水に溶けてモデルにきれいな表面を残します。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 180–220 °C |
| ベッド温度 | 35–60 °C |
| 乾燥 | 重要（55 °C / 6–8時間） |

:::danger PVAは極めて吸湿性が高い
PVAは他のどの一般的なフィラメントよりも速く湿気を吸収します。プリント前にPVAを十分に乾燥させ、シリカゲルの入った密封ボックスに常に保管してください。湿ったPVAはノズルに詰まり取り除くのが非常に困難です。
:::

### 使用と溶解

1. PVAをサポート素材として使ってモデルを印刷（マルチマテリアルプリンター — AMSが必要）
2. 完成した印刷物を温水（30–40 °C）に浸ける
3. 30–120分放置し、必要に応じて水を交換
4. 清水で洗い流して乾燥させる

可能であれば**PVA専用エクストルーダーを使用** — 標準エクストルーダーのPVA残留物は次の印刷を台無しにする可能性があります。

### 用途
- 手動で取り除けない複雑なサポート構造
- 目立つ表面跡のない内部オーバーハングサポート
- 空洞と内部チャネルを持つモデル

---

## HIPS（高耐衝撃ポリスチレン）— 溶剤溶解性サポート素材

HIPSはABSと一緒に使用するよう設計されたもう一つのサポート素材です。**リモネン**（柑橘類系溶剤）に溶けます。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 220–240 °C |
| ベッド温度 | 90–110 °C |
| チャンバー温度 | 45–55 °C |
| 乾燥 | 推奨（65 °C / 4–6時間） |

### ABSとの使用

HIPSはABSと同じ温度で印刷され、よく接着します。印刷後、HIPSはD-リモネンに30–60分浸けることで溶けます。

:::warning リモネンは水ではありません
D-リモネンはオレンジの皮から抽出された溶剤です。比較的無害ですが、手袋を使用し換気された場所で作業してください。使用済みリモネンを下水に捨てないでください — 廃棄物処理センターに持参してください。
:::

### 比較：PVA対HIPS

| 特徴 | PVA | HIPS |
|------|-----|------|
| 溶剤 | 水 | D-リモネン |
| 対応素材 | PLA互換 | ABS互換 |
| 湿気感受性 | 極めて高い | 中程度 |
| コスト | 高い | 中程度 |
| 入手可能性 | 良好 | 中程度 |

## PVB / Fibersmooth — エタノール平滑化可能素材

PVB（ポリビニルブチラール）はアセトンでABSを平滑化するのと同じように、はるかに安全に**エタノール（アルコール）で平滑化**できるユニークな素材です。

### 設定

| パラメータ | 値 |
|-----------|-----|
| ノズル温度 | 190–210 °C |
| ベッド温度 | 35–55 °C |
| 部品冷却 | 80–100% |
| 乾燥 | 推奨（55 °C / 4時間） |

### エタノール平滑化

1. 標準PVB設定でモデルを印刷
2. 筆で99%イソプロピルアルコール（IPA）またはエタノールを塗布
3. 10–15分乾燥させる — 表面が均一に流れ出る
4. より滑らかな表面のために必要に応じて繰り返す
5. 代替：塗布して5分間密封容器に入れて蒸気処理

:::tip アセトンより安全
IPA/エタノールはアセトンよりはるかに安全に取り扱えます。引火点が高く蒸気の毒性がはるかに低いです。それでも良好な換気を推奨します。
:::

### 用途
- 滑らかな表面が望まれるフィギュアと装飾
- 展示するプロトタイプ
- 塗装する部品 — 滑らかな表面でより良い塗装

## 特殊素材の推奨造形プレート

| 素材 | 推奨プレート | のりスティック？ |
|------|------------|---------------|
| ASA | Engineering Plate / High Temp Plate | はい |
| PC | High Temp Plate | はい（必須） |
| PP | Engineering Plate + PPテープ | PP専用テープ |
| PVA | Cool Plate / Textured PEI | いいえ |
| HIPS | Engineering Plate / High Temp Plate | はい |
| PVB | Cool Plate / Textured PEI | いいえ |
