# 素材比較

適切なフィラメントを選ぶことは、仕事に適切なツールを選ぶのと同じくらい重要です。この記事では全体像を提供します — シンプルな比較表からショア硬度、HDT値、実践的な意思決定ガイドまで。

## 適切な素材の選択 — 意思決定ガイド

何を選べばいいかわからない？以下の質問に従ってください：

### 熱に耐える必要があるか？
**はい** → ABS、ASA、PC、またはPA

- 少し熱い（~90 °Cまで）：**ABS** または **ASA**
- 非常に熱い（100 °C以上）：**PC** または **PA**
- 最大耐温性：**PC**（~120 °Cまで）または **PA-CF**（~160 °Cまで）

### 柔軟性が必要か？
**はい** → **TPU**

- 非常に柔らかい（ゴムのような）：TPU 85A
- 標準的な柔軟性：TPU 95A
- セミフレキシブル：PETGまたはPA

### 屋外で使用するか？
**はい** → **ASA** が明確な選択

ASAはUV露出専用に開発されており、屋外ではABSより優れています。ASAが入手できない場合はPETGが次善の選択です。

### 最大強度が必要か？
**はい** → **PA-CF** または **PC**

- 最強の軽量複合材：**PA-CF**
- 最強の純熱可塑性プラスチック：**PC**
- 低コストで良好な強度：**PA（ナイロン）**

### 最も簡単な印刷？
→ **PLA**

PLAは最も許容度の高い素材です。最低温度、エンクロージャー不要、反り要求最小。

### 食品接触？
→ **PLA**（但し条件あり）

PLA自体は無毒ですが：
- ステンレス鋼ノズルを使用（真鍮ではなく — 鉛を含む可能性がある）
- FDMプリントは多孔質表面のため「食品安全」ではない — バクテリアが繁殖できる
- 過酷な環境（酸性、熱水、食洗機）を避ける
- PETGは使い捨て接触の方が良い代替品

## ショア硬度の説明

ショア硬度はエラストマーとプラスチック素材の硬さと剛性を説明するために使用されます。3Dプリントでは特にTPUや他のフレキシブルフィラメントに関連します。

### ショアA — フレキシブル素材

ショアAスケールは0（極めて柔らかい、ほぼゲル状）から100（極めて硬いゴム）まであります。90Aを超える値はリジッドなプラスチック素材に近づき始めます。

| ショアA値 | 体感硬度 | 例 |
|---|---|---|
| 30A | 極めて柔らかい | シリコン、ゼリー |
| 50A | 非常に柔らかい | 柔らかいゴム、耳栓 |
| 70A | 柔らかい | 車のホース、ランニングシューズのミッドソール |
| 85A | 中程度の柔らかさ | 自転車タイヤ、柔らかいTPUフィラメント |
| 95A | セミリジッド | 標準TPUフィラメント |
| 100A ≈ 55D | スケール間の境界 | — |

**TPU 95A**は3Dプリントの業界標準で、弾性と印刷性の良いバランスを提供します。**TPU 85A**は非常に柔らかく、印刷時に多くの忍耐が必要です。

### ショアD — リジッド素材

ショアDはより硬い熱可塑性プラスチックに使用されます：

| 素材 | おおよそのショアD |
|---|---|
| PLA | ~80D |
| PETG | ~70D |
| ABS | ~75D |
| PC | ~80D |
| PA（ナイロン） | ~70–75D |

:::tip 同じではありません
ショアA 95とショアD 40は数字が近いように見えても同じではありません。スケールは異なり、100A/55D境界付近でのみ部分的に重なります。サプライヤーが示すスケールを常に確認してください。
:::

## 温度許容範囲 — HDTとVST

素材が変形し始める温度を知ることは機能部品に重要です。2つの標準測定が使用されます：

- **HDT（熱変形温度）** — 素材が標準化された荷重下で0.25mm曲がる温度。荷重下での使用温度の尺度。
- **VST（ビカット軟化温度）** — 標準化された針が素材に1mm沈む温度。荷重なしの絶対軟化点の尺度。

| 素材 | HDT（°C） | VST（°C） | 実用最大温度 |
|---|---|---|---|
| PLA | 52–60 | 55–65 | ~50 °C |
| PETG | 70–80 | 75–85 | ~70 °C |
| ABS | 85–105 | 95–110 | ~90 °C |
| ASA | 90–105 | 95–108 | ~90 °C |
| TPU | 40–70 | 変動 | ~60 °C |
| PA（ナイロン） | 70–180 | 180–220 | ~80–160 °C |
| PA-CF | 100–200 | 200–230 | ~100–180 °C |
| PC | 120–140 | 145–160 | ~120 °C |
| PLA-CF | 55–65 | 60–70 | ~55 °C |

:::warning 暖かい環境でのPLA
夏の車内のPLA部品は災害のレシピです。駐車した車のダッシュボードは80–90 °Cに達する可能性があります。日光や熱にさらされる可能性があるものにはABS、ASA、またはPETGを使用してください。
:::

:::info PA変種は特性が非常に異なります
PAは単一の素材ではなく素材ファミリーです。PA6のHDTは低く（~70 °C）、PA12とPA6-CFは160–200 °Cになります。使用するフィラメントのデータシートを常に確認してください。
:::

## ノズル要件

### 真鍮ノズル（標準）

カーボンファイバーまたはガラスファイバー充填**なし**のすべての素材に対応：
- PLA、PETG、ABS、ASA、TPU、PA、PC、PVA
- 真鍮は柔らかく、研磨性の高い素材で急速に摩耗する

### 焼入れ鋼ノズル（複合材に必須）

**必須**：
- PLA-CF（カーボンファイバーPLA）
- PETG-CF
- PA-CF
- ABS-GF（ガラスファイバーABS）
- PPA-CF、PPA-GF
- 名前に「-CF」、「-GF」、「-HF」または「カーボンファイバー」を含むすべてのフィラメント

焼入れ鋼は真鍮より熱伝導率が低い — ノズル温度を+5–10 °C補正してください。

:::danger カーボンファイバーフィラメントは真鍮ノズルを急速に損傷します
真鍮ノズルはわずか数百グラムのCFフィラメントで顕著に摩耗することがあります。結果は徐々に悪化するアンダー押出と不正確な寸法です。複合材を印刷する場合は焼入れ鋼に投資してください。
:::

## 用途別素材の簡単な概要

| 用途 | 推奨素材 | 代替 |
|---|---|---|
| 装飾、フィギュア | PLA、PLA Silk | PETG |
| 機能的な室内部品 | PETG | PLA+ |
| 屋外露出 | ASA | PETG |
| フレキシブル部品、ケース | TPU 95A | TPU 85A |
| エンジンルーム、熱い環境 | PA-CF、PC | ABS |
| 軽量で剛性のある構造 | PLA-CF | PA-CF |
| サポート素材（溶解性） | PVA | HIPS |
| 食品接触（限定） | PLA（ステンレスノズル） | — |
| 最大強度 | PA-CF | PC |
| 透明 | クリアPETG | クリアPC |
