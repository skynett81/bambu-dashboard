# Bambu Dashboardへようこそ

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/V7V21NRKM7)

**Bambu Dashboard**は、Bambu Lab 3Dプリンター向けのセルフホスト型フル機能コントロールパネルです。プリンター、フィラメント在庫、印刷履歴などをすべてひとつのブラウザタブから完全に把握・管理できます。

## Bambu Dashboardとは？

Bambu DashboardはMQTT over LANを通じてプリンターに直接接続し、Bambu Labのサーバーに依存しません。モデルや印刷履歴の同期のためにBambu Cloudに接続することも可能です。

### 主な機能

- **ライブダッシュボード** — リアルタイム温度、進行状況、カメラ、AMS状態（LIVEインジケーター付き）
- **フィラメント在庫** — AMS同期、EXTスプール対応、素材情報、プレート互換性、乾燥ガイド付きですべてのスプールを管理
- **フィラメントトラッキング** — 4段階フォールバックによる正確なトラッキング（AMSセンサー → EXT推定 → クラウド → 時間）
- **素材ガイド** — 15種類の素材について温度、プレート互換性、乾燥、特性、ヒントを掲載
- **印刷履歴** — モデル名、MakerWorldリンク、フィラメント使用量、コストを含む完全なログ
- **プランナー** — カレンダー表示、負荷分散とフィラメントチェック付き印刷キュー
- **プリンター制御** — 温度、速度、ファン、G-codeコンソール
- **Print Guard** — xcam + 5つのセンサーモニターによる自動保護
- **コスト見積もり** — 素材、電力、作業、消耗、マークアップ、販売価格提案
- **メンテナンス** — KBベースの間隔、ノズル寿命、プレート寿命、ガイド付きトラッキング
- **サウンドアラート** — カスタム音声アップロードとプリンタースピーカー（M300）対応の9つの設定可能なイベント
- **アクティビティログ** — すべてのイベント（印刷、エラー、メンテナンス、フィラメント）の永続タイムライン
- **通知** — 7チャンネル（Telegram、Discord、メール、ntfy、Pushover、SMS、webhook）
- **マルチプリンター** — Bambu Labシリーズ全体に対応
- **17言語** — ノルウェー語、英語、ドイツ語、フランス語、スペイン語、イタリア語、日本語、韓国語、オランダ語、ポーランド語、ポルトガル語、スウェーデン語、トルコ語、ウクライナ語、中国語、チェコ語、ハンガリー語
- **セルフホスト** — クラウド依存なし、データは自分のマシンに

### v1.1.13の新機能

- **EXTスプール検出** — MQTTマッピングフィールドによるP2S/A1対応、外部スプールのフィラメント使用量を正確にトラッキング
- **フィラメント素材データベース** — 15種類の素材、プレート互換性、乾燥ガイド、特性
- **メンテナンスパネル** — KBベースの間隔、4種類の新しいノズルタイプ、ドキュメントリンク付きガイドタブ
- **サウンドアラート** — 9イベント、カスタムアップロード（MP3/OGG/WAV、最大10秒）、音量調整、プリンタースピーカー
- **アクティビティログ** — ページが開いていなくてもすべてのデータベースからの永続タイムライン
- **HMSエラーコード** — 270以上のコードの読みやすい説明
- **完全なi18n** — 17言語に翻訳された2944すべてのキー
- **自動ビルドドキュメント** — インストールとサーバー起動時に自動でドキュメントをビルド

## 対応プリンター

LANモード対応のすべてのBambu Labプリンター：

- **X1シリーズ**: X1C、X1C Combo、X1E
- **P1シリーズ**: P1S、P1S Combo、P1P
- **P2シリーズ**: P2S、P2S Combo
- **Aシリーズ**: A1、A1 Combo、A1 mini
- **H2シリーズ**: H2S、H2D（デュアルノズル）、H2C（ツールチェンジャー、6ヘッド）

## 機能の詳細

### フィラメントトラッキング

ダッシュボードは4段階フォールバックで自動的にフィラメント使用量をトラッキングします：

1. **AMSセンサー差分** — 最も正確、開始/終了の残量%を比較
2. **EXT直接** — vt_trayなしのP2S/A1用、クラウド推定を使用
3. **クラウド推定** — Bambu Cloud印刷ジョブデータから
4. **時間推定** — 最終フォールバックとして約30g/時間

失敗した印刷後のエラーを避けるため、すべての値はAMSセンサーとスプールデータベースの最小値として表示されます。

### 素材ガイド

15種類の素材を含む組み込みデータベース：
- 温度（ノズル、ベッド、チャンバー）
- プレート互換性（Cool、Engineering、High Temp、Textured PEI）
- 乾燥情報（温度、時間、吸湿性）
- 8つの特性（強度、柔軟性、耐熱性、UV、表面、使いやすさ）
- 難易度と特別な要件（硬化ノズル、エンクロージャー）

### サウンドアラート

以下に対応した9つの設定可能なイベント：
- **カスタム音声クリップ** — MP3/OGG/WAVをアップロード（最大10秒、500 KB）
- **組み込みトーン** — Web Audio APIで生成されたメタリック/シンセサウンド
- **プリンタースピーカー** — プリンターのブザーに直接M300 G-codeメロディー
- **カウントダウン** — 印刷残り1分時のサウンドアラート

### メンテナンス

完全なメンテナンスシステム：
- コンポーネントトラッキング（ノズル、PTFEチューブ、ロッド、ベアリング、AMS、プレート、乾燥）
- ドキュメントからのKBベースの間隔
- タイプ別ノズル寿命（真鍮、硬化鋼、HS01）
- タイプ別プレート寿命（Cool、Engineering、High Temp、Textured PEI）
- 完全なドキュメントへのヒントとリンク付きガイドタブ

## 技術概要

Bambu DashboardはNode.js 22とバニラHTML/CSS/JSで構築されており、重いフレームワークやビルドステップは不要です。データベースはNode.js 22に組み込まれたSQLiteです。

- **バックエンド**: Node.js 22、npmパッケージは3つのみ（mqtt、ws、basic-ftp）
- **フロントエンド**: バニラHTML/CSS/JS、ビルドステップなし
- **データベース**: Node.js 22組み込みの`--experimental-sqlite`経由のSQLite
- **ドキュメント**: 17言語対応のDocusaurus、インストール時に自動ビルド
- **API**: 177以上のエンドポイント、`/api/docs`にOpenAPIドキュメント

詳細は[アーキテクチャ](./avansert/arkitektur)を参照してください。
