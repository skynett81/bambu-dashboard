# Сушіння філаменту

Вологий філамент є однією з найпоширеніших та найнедооцінених причин поганих друків. Навіть філамент, який виглядає сухим, міг увібрати достатньо вологи, щоб зіпсувати результат — особливо для таких матеріалів, як нейлон та PVA.

## Навіщо сушити філамент?

Багато видів пластику є **гігроскопічними** — вони поглинають вологу з повітря з часом. Коли вологий філамент проходить через гарячу сопло, вода раптово випаровується та створює мікробульбашки в розплавленому пластику. Результат:

- **Тріскотіння та клацання** під час друку
- **Туман або пара**, видима з сопла
- **Stringing та hairing**, які не усуваються регулюваннями
- **Шорстка або зерниста поверхня** — особливо на верхніх шарах
- **Слабкі деталі** з поганою адгезією шарів та мікротріщинами
- **Матове або тьмяне покриття** на матеріалах, які зазвичай повинні бути блискучими або шовковистими

:::warning Сушіть філамент ДО регулювання налаштувань
Багато людей витрачають години на налаштування ретракції та температури без покращення — тому що причиною є вологий філамент. Завжди сушіть філамент та повторно тестуйте перед зміною налаштувань друку.
:::

## Які матеріали потребують сушіння?

Всі види пластику можуть зволожуватися, але ступінь гігроскопічності значно варіюється:

| Матеріал | Гігроскопічність | Температура сушіння | Час сушіння | Пріоритет |
|---|---|---|---|---|
| PLA | Низька | 45–50 °C | 4–6 годин | Необов'язково |
| PETG | Середня | 65 °C | 4–6 годин | Рекомендовано |
| ABS | Середня | 65–70 °C | 4 години | Рекомендовано |
| TPU | Середня | 50–55 °C | 4–6 годин | Рекомендовано |
| ASA | Середня | 65 °C | 4 години | Рекомендовано |
| PC | Висока | 70–80 °C | 6–8 годин | Необхідно |
| PA/Нейлон | Надзвичайно висока | 70–80 °C | 8–12 годин | НЕОБХІДНО |
| PA-CF | Надзвичайно висока | 70–80 °C | 8–12 годин | НЕОБХІДНО |
| PVA | Надзвичайно висока | 45–50 °C | 4–6 годин | НЕОБХІДНО |

:::tip Нейлон та PVA критичні
PA/Нейлон та PVA можуть поглинути достатньо вологи, щоб стати непридатними для друку за **кілька годин** у нормальному кліматі приміщення. Ніколи не відкривайте нову котушку цих матеріалів без негайного сушіння — і завжди друкуйте з закритого контейнера або сушарки.
:::

## Ознаки вологого філаменту

Не завжди потрібно сушити філамент за таблицею. Навчіться розпізнавати симптоми:

| Симптом | Вологість? | Інші можливі причини |
|---|---|---|
| Тріскіт/клацання | Так, дуже ймовірно | Частково забита сопло |
| Туман/пара з сопла | Так, майже напевно | Немає інших причин |
| Шорстка, зерниста поверхня | Так, можливо | Занадто низька температура, занадто висока швидкість |
| Stringing, що не зникає | Так, можливо | Неправильна ретракція, занадто висока температура |
| Слабкі, крихкі деталі | Так, можливо | Занадто низька температура, неправильне заповнення |
| Зміна кольору або матове покриття | Так, можливо | Неправильна температура, спалений пластик |

## Методи сушіння

### Сушарка для філаменту (рекомендовано)

Спеціальні сушарки для філаменту є найпростішим та найбезпечнішим рішенням. Вони підтримують точну температуру та дозволяють друкувати безпосередньо з сушарки протягом усього завдання.

Популярні моделі:
- **eSun eBOX** — доступна ціна, можна друкувати з коробки, підтримує більшість матеріалів
- **Bambu Lab Filament Dryer** — оптимізована для Bambu AMS, підтримує високі температури
- **Polymaker PolyDryer** — хороший термометр та хороший контроль температури
- **Sunlu S2 / S4** — бюджетний варіант, кілька котушок одночасно

Процедура:
```
1. Помістіть котушки в сушарку
2. Встановіть температуру з таблиці вище
3. Встановіть таймер на рекомендований час
4. Зачекайте — не переривайте процес передчасно
5. Друкуйте безпосередньо з сушарки або одразу запакуйте
```

### Харчовий дегідратор

Звичайний харчовий дегідратор напрочуд добре працює як сушарка для філаменту:

- Доступна ціна при покупці
- Хороша циркуляція повітря
- Підтримує температури до 70–75 °C на багатьох моделях

:::warning Перевірте максимальну температуру вашого дегідратора
Багато дешевих харчових дегідраторів мають неточні термостати та можуть відхилятися ±10 °C. Для PA та PC, які потребують високої температури, виміряйте фактичну температуру термометром.
:::

### Духовка

Кухонну духовку можна використовувати в крайньому випадку, але це вимагає обережності:

:::danger НІКОЛИ не використовуйте звичайну духовку вище 60 °C для PLA — деформується!
PLA починає розм'якшуватися вже при 55–60 °C. Гаряча духовка може пошкодити котушки, розплавити серцевину та зробити філамент непридатним. Ніколи не використовуйте духовку для PLA, якщо ви не впевнені, що температура точно відкалібрована та нижче 50 °C.
:::

Для матеріалів, що витримують вищі температури (ABS, ASA, PA, PC):
```
1. Попередньо нагрійте духовку до бажаної температури
2. Використовуйте термометр для перевірки фактичної температури
3. Покладіть котушки на решітку (не безпосередньо на дно духовки)
4. Залиште двері прочиненими для виходу вологи
5. Стежте при першому використанні цього методу
```

### Bambu Lab AMS

Bambu Lab AMS Lite та AMS Pro мають вбудовану функцію сушіння (низька температура + циркуляція повітря). Це не замінює повне сушіння, але підтримує вже висушений філамент сухим під час друку.

- AMS Lite: Пасивне сушіння — обмежує поглинання вологи, не сушить активно
- AMS Pro: Активний нагрів — певне сушіння можливе, але не таке ефективне, як спеціальна сушарка

## Зберігання філаменту

Правильне зберігання після сушіння так само важливе, як і сам процес сушіння:

### Найкращі рішення

1. **Сухий шафа з силікагелем** — спеціальна шафа з гігрометром та осушувачем. Підтримує вологість стабільно низькою (ідеально нижче 20% RH)
2. **Вакуумні пакети** — відкачайте повітря та запечатайте з осушувачем всередині. Найдешевше довгострокове зберігання
3. **Пакети Ziplock з осушувачем** — просто та ефективно для коротших періодів

### Силікагель та осушувачі

- **Синій/помаранчевий силікагель** вказує на ступінь насичення — замініть або регенеруйте (сушіть у духовці при 120 °C) при зміні кольору
- **Гранульований силікагель** ефективніший за порошкоподібний
- **Пакети з осушувачем** від виробників філаменту можна регенерувати та використовувати повторно

### Гігрометр у коробці для зберігання

Дешевий цифровий гігрометр показує поточну вологість повітря в коробці:

| Відносна вологість (RH) | Статус |
|---|---|
| Нижче 15% | Ідеально |
| 15–30% | Добре для більшості матеріалів |
| 30–50% | Прийнятно для PLA та PETG |
| Вище 50% | Проблематично — особливо для PA, PVA, PC |

## Практичні поради

- **Сушіть безпосередньо ПЕРЕД друком** — висушений філамент може зволожитися знову за дні у нормальному кліматі приміщення
- **Друкуйте з сушарки** для PA, PC та PVA — не просто сушіть та відкладайте
- **Нова котушка ≠ суха котушка** — виробники запечатують з осушувачем, але ланцюжок зберігання міг підвести. Завжди сушіть нові котушки гігроскопічних матеріалів
- **Позначайте висушені котушки** датою сушіння
- **Спеціальна PTFE-трубка** від сушарки до принтера мінімізує вплив вологи під час друку

## Bambu Dashboard та статус сушіння

Bambu Dashboard дозволяє реєструвати інформацію про філамент, включаючи дату останнього сушіння, у профілях філаменту. Використовуйте це для відстеження того, які котушки свіжовисушені, а які потребують нового сушіння.
