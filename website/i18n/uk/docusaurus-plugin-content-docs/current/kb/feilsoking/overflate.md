# Дефекти поверхні

Поверхня 3D-друку багато розповідає про те, що відбувається всередині системи. Більшість дефектів поверхні мають одну-дві чіткі причини — за правильної діагностики вони напрочуд легко усуваються.

## Швидкий огляд діагностики

| Симптом | Найпоширеніша причина | Перший крок |
|---|---|---|
| Blobs та zits | Надекструзія, розміщення шва | Налаштуйте шов, відкалібруйте потік |
| Видимі лінії шарів | Z-wobble, занадто товсті шари | Перейдіть на тонші шари, перевірте вісь Z |
| Elephant foot | Перший шар занадто широкий | Компенсація elephant foot |
| Ringing/ghosting | Вібрації на великій швидкості | Знизьте швидкість, увімкніть input shaper |
| Недоекструзія | Забита сопло, занадто низька температура | Очистіть сопло, підвищте температуру |
| Надекструзія | Занадто висока швидкість потоку | Відкалібруйте швидкість потоку |
| Pillowing | Мало верхніх шарів, мало охолодження | Збільшіть кількість верхніх шарів, посильте охолодження |
| Розшарування | Занадто низька температура, забагато охолодження | Підвищте температуру, зменшіть вентилятор |

## Blobs та zits

Blobs — це неправильні грудки на поверхні. Zits — це точкоподібні точки, часто вздовж лінії шва.

### Причини

- **Надекструзія** — занадто багато пластику видавлюється та відтискається вбік
- **Погане розміщення шва** — стандартний шов «nearest» збирає всі переходи в одному місці
- **Проблема ретракції** — недостатня ретракція створює накопичення тиску у соплі
- **Вологий філамент** — вологість створює мікробульбашки та крапельки

### Рішення

**Налаштування шва в Bambu Studio:**
```
Bambu Studio → Якість → Позиція шва
- Aligned: Усі шви в одному місці (видимо, але охайно)
- Nearest: Найближча точка (розподіляє blobs випадково)
- Back: За об'єктом (рекомендовано для візуальної якості)
- Random: Випадковий розподіл (найкраще маскує шов)
```

**Калібрування швидкості потоку:**
```
Bambu Studio → Калібрування → Швидкість потоку
Регулюйте кроками ±2% поки blobs не зникнуть
```

:::tip Шов у позиції «Back» для візуальної якості
Розмістіть шов ззаду об'єкта, щоб він був якнайменш видимим. Поєднайте з «Wipe on retract» для чистішого завершення шва.
:::

## Видимі лінії шарів

Усі FDM-друки мають лінії шарів, але вони повинні бути рівномірними та ледь помітними при нормальних друках. Аномальна видимість вказує на конкретні проблеми.

### Причини

- **Z-wobble** — вісь Z вібрує або не рівна, створює хвилясту картину по всій висоті
- **Занадто товсті шари** — висота шару понад 0,28 мм помітна навіть при ідеальних друках
- **Коливання температури** — непослідовна температура плавлення дає мінливу ширину шарів
- **Непослідовний діаметр філаменту** — дешевий філамент зі змінним діаметром

### Рішення

**Z-wobble:**
- Перевірте, чи шпиндель (Z-leadscrew) чистий та змащений
- Перевірте, чи шпиндель не зігнутий (візуальний огляд при обертанні)
- Дивіться статтю про обслуговування [змащення осі Z](/docs/kb/vedlikehold/smoring)

**Висота шару:**
- Перейдіть на 0,12 мм або 0,16 мм для більш рівномірної поверхні
- Пам'ятайте, що зменшення висоти вдвічі подвоює час друку

**Коливання температури:**
- Використовуйте PID-калібрування (доступне через меню обслуговування Bambu Studio)
- Уникайте протягів, які охолоджують сопло під час друку

## Elephant foot

Elephant foot — це ситуація, коли перший шар ширший за решту об'єкта, ніби об'єкт «розкривається» знизу.

### Причини

- Перший шар надто сильно притискується до платформи (Z-offset занадто близько)
- Занадто висока температура столу тримає пластик м'яким та рідким надто довго
- Недостатнє охолодження першого шару дає пластику більше часу на розтікання

### Рішення

**Компенсація elephant foot у Bambu Studio:**
```
Bambu Studio → Якість → Компенсація Elephant foot
Починайте з 0,1–0,2 мм та регулюйте, поки нога не зникне
```

**Z-offset:**
- Заново відкалібруйте висоту першого шару
- Підвищуйте Z-offset на 0,05 мм за раз, поки нога не зникне

**Температура столу:**
- Знизьте температуру столу на 5–10 °C
- Для PLA: 55 °C зазвичай достатньо — 65 °C може спричиняти elephant foot

:::warning Не компенсуйте надмірно
Занадто висока компенсація elephant foot може створити зазор між першим шаром та рештою. Регулюйте обережно кроками 0,05 мм.
:::

## Ringing та ghosting

Ringing (також «ghosting» або «echoing») — це хвилястий візерунок на поверхні відразу після гострих країв або кутів. Візерунок «відлунює» від краю.

### Причини

- **Вібрації** — швидке прискорення та уповільнення на кутах надсилає вібрації через раму
- **Занадто висока швидкість** — особливо швидкість зовнішньої стінки понад 100 мм/с дає помітний ringing
- **Слабкі деталі** — розхитана котушка, вібруючий кабелепровід або неміцно встановлений принтер

### Рішення

**Bambu Lab input shaper (Компенсація резонансу):**
```
Bambu Studio → Принтер → Компенсація резонансу
Bambu Lab X1C та P1S мають вбудований акселерометр та калібрують це автоматично
```

**Знизьте швидкість:**
```
Зовнішня стінка: знизьте до 60–80 мм/с
Прискорення: знизьте зі стандартного до 3000–5000 мм/с²
```

**Механічна перевірка:**
- Перевірте, чи принтер стоїть на стабільній поверхні
- Перевірте, чи котушка не вібрує в тримачі
- Затягніть усі доступні гвинти на зовнішніх панелях рами

:::tip X1C та P1S автоматично калібрують ringing
Bambu Lab X1C та P1S мають вбудоване калібрування акселерометра, яке запускається автоматично при старті. Запустіть «Повне калібрування» з меню обслуговування, якщо ringing з'явиться через деякий час.
:::

## Недоекструзія

Недоекструзія — це ситуація, коли принтер видавлює занадто мало пластику. Результат — тонкі, слабкі стінки, видимі зазори між шарами та «розтріпана» поверхня.

### Причини

- **Частково забите сопло** — накопичення вуглецю зменшує прохідність
- **Занадто низька температура сопла** — пластик недостатньо рідкий
- **Зношена шестерня** в механізмі екструдера не схоплює філамент достатньо добре
- **Занадто висока швидкість** — екструдер не встигає за бажаним потоком

### Рішення

**Холодне витягування (Cold pull):**
```
1. Нагрійте сопло до 220 °C
2. Вручну подайте філамент
3. Охолодіть сопло до 90 °C (PLA), утримуючи тиск
4. Швидко витягніть філамент
5. Повторюйте, поки матеріал, що витягується, не стане чистим
```

**Регулювання температури:**
- Підвищте температуру сопла на 5–10 °C та повторіть тест
- Занадто низька температура — поширена причина недоекструзії

**Калібрування швидкості потоку:**
```
Bambu Studio → Калібрування → Швидкість потоку
Поступово збільшуйте, поки недоекструзія не зникне
```

**Перевірте шестерню екструдера:**
- Витягніть філамент та огляньте шестерню
- Очистіть невеликою щіткою, якщо в зубцях є пил від філаменту

## Надекструзія

Надекструзія дає занадто широку нитку — поверхня виглядає рихлою, блискучою або нерівномірною, зі схильністю до blobs.

### Причини

- **Занадто висока швидкість потоку** (EM — Extrusion Multiplier)
- **Неправильний діаметр філаменту** — філамент 2,85 мм з профілем 1,75 мм дає масову надекструзію
- **Занадто висока температура сопла** робить пластик надто рідким

### Рішення

**Калібрування швидкості потоку:**
```
Bambu Studio → Калібрування → Швидкість потоку
Знижуйте кроками 2% поки поверхня не стане рівномірною та матовою
```

**Перевірте діаметр філаменту:**
- Виміряйте фактичний діаметр філаменту штангенциркулем у 5–10 місцях вздовж філаменту
- Середнє відхилення понад 0,05 мм вказує на дешевий філамент

## Pillowing

Pillowing — це нерівні, горбкуваті верхні шари з «подушками» пластику між ними. Особливо помітно при низькому заповненні та малій кількості верхніх шарів.

### Причини

- **Мало верхніх шарів** — пластик над заповненням провалюється в отвори
- **Мало охолодження** — пластик не затвердіває достатньо швидко, щоб перекинути місток над отворами заповнення
- **Занадто низьке заповнення** — великі отвори у заповненні важко перекрити

### Рішення

**Збільшіть кількість верхніх шарів:**
```
Bambu Studio → Якість → Верхні шари оболонки
Мінімум: 4 шари
Рекомендовано для рівномірної поверхні: 5–6 шарів
```

**Посильте охолодження:**
- PLA повинен мати вентилятор охолодження на 100% з 3-го шару
- Недостатнє охолодження — найпоширеніша причина pillowing

**Збільшіть заповнення:**
- Перейдіть з 10–15% до 20–25%, якщо pillowing не зникає
- Візерунок Gyroid дає більш рівномірну поверхню місткаа, ніж лінії

:::tip Прасування (Ironing)
Функція «ironing» у Bambu Studio проводить сопло над верхнім шаром ще раз для вирівнювання поверхні. Увімкніть у Якість → Ironing для найкращого фінішу верхнього шару.
:::

## Розшарування (Layer separation / delamination)

Розшарування — це ситуація, коли шари не прилипають один до одного належним чином. У найгіршому випадку друк розтріскується вздовж ліній шарів.

### Причини

- **Занадто низька температура сопла** — пластик не плавиться достатньо добре в нижній шар
- **Забагато охолодження** — пластик застигає надто швидко, перш ніж встигне злитися
- **Занадто велика товщина шару** — понад 80% діаметра сопла дає погану адгезію
- **Занадто висока швидкість** — зменшений час плавлення на мм траєкторії

### Рішення

**Підвищте температуру:**
- Спробуйте +10 °C від стандарту та спостерігайте
- ABS та ASA особливо чутливі — потребують контрольованого обігріву камери

**Зменшіть охолодження:**
- ABS: вентилятор ВИМКНЕНИЙ (0%)
- PETG: максимум 20–40%
- PLA: може витримувати повне охолодження, але зменшіть, якщо виникає розшарування

**Товщина шару:**
- Використовуйте максимум 75% діаметра сопла
- З соплом 0,4 мм: максимальна рекомендована висота шару 0,30 мм

**Перевірте, чи камера достатньо тепла (ABS/ASA/PA/PC):**
```
Bambu Lab X1C/P1S: Дайте камері прогрітися до 40–60 °C
перед початком друку — не відкривайте двері під час друку
```

## Загальні поради з усунення несправностей

1. **Змінюйте одну річ за раз** — тестуйте невеликим калібрувальним друком між кожною зміною
2. **Спочатку висушіть філамент** — багато дефектів поверхні — насправді симптоми вологості
3. **Очистіть сопло** — часткове засмічення дає непослідовні дефекти поверхні, які важко діагностувати
4. **Проведіть повне калібрування** з меню обслуговування Bambu Studio після великих коригувань
5. **Використовуйте Bambu Dashboard** для відстеження того, які налаштування давали найкращий результат з часом
