## Виберіть правильний матеріал — посібник з прийняття рішень

Не впевнені, що обрати? Слідкуйте за цими питаннями:

### Чи потрібно витримувати тепло?
**Так** → ABS, ASA, PC або PA

- Трохи тепла (до ~90 °C): **ABS** або **ASA**
- Багато тепла (понад 100 °C): **PC** або **PA**
- Максимальна термостійкість: **PC** (до ~120 °C) або **PA-CF** (до ~160 °C)

### Чи потрібна гнучкість?
**Так** → **TPU**

- Дуже м'який (як гума): TPU 85A
- Стандартно гнучкий: TPU 95A
- Напівгнучкий: PETG або PA

### Чи буде використовуватися на вулиці?
**Так** → **ASA** — очевидний вибір

ASA розроблений спеціально для UV-впливу та перевершує ABS на вулиці. PETG — другий найкращий вибір, якщо ASA недоступний.

### Чи потрібна максимальна міцність?
**Так** → **PA-CF** або **PC**

- Найміцніший легкий композит: **PA-CF**
- Найміцніший чистий термопластик: **PC**
- Хороша міцність за нижчою ціною: **PA (Нейлон)**

### Друк максимально просто?
→ **PLA**

PLA — найпрощальніший матеріал з існуючих. Найнижча температура, без вимог до камери, низький ризик жолоблення.

### Контакт з їжею?
→ **PLA** (з застереженнями)

PLA сам по собі нетоксичний, але:
- Використовуйте сопло з нержавіючої сталі (не латунне — може містити свинець)
- Друки FDM ніколи не є «безпечними для їжі» через пористу поверхню — бактерії можуть рости
- Уникайте складних умов (кислоти, гаряча вода, посудомийна машина)
- PETG — краща альтернатива для одноразового контакту

## Твердість Shore пояснена

Твердість Shore використовується для опису твердості та жорсткості еластомерів та пластикових матеріалів. Для 3D-друку вона особливо актуальна для TPU та інших гнучких філаментів.

### Shore A — гнучкі матеріали

Шкала Shore A іде від 0 (надзвичайно м'який, майже як гель) до 100 (надзвичайно тверда гума). Значення понад 90A починають наближатися до жорстких пластикових матеріалів.

| Значення Shore A | Відчутна твердість | Приклад |
|---|---|---|
| 30A | Надзвичайно м'який | Силікон, желе |
| 50A | Дуже м'який | М'яка гума, навушники |
| 70A | М'який | Автомобільна шина, підошва кросівки |
| 85A | Середньо м'який | Велосипедна шина, м'які TPU-філаменти |
| 95A | Напівжорсткий | Стандартний TPU-філамент |
| 100A ≈ 55D | Межа між шкалами | — |

**TPU 95A** є галузевим стандартом для 3D-друку та забезпечує хороший баланс між еластичністю та придатністю для друку. **TPU 85A** дуже м'який та вимагає більше терпіння під час друку.

### Shore D — жорсткі матеріали

Shore D використовується для твердіших термопластиків:

| Матеріал | Приблизний Shore D |
|---|---|
| PLA | ~80D |
| PETG | ~70D |
| ABS | ~75D |
| PC | ~80D |
| PA (Нейлон) | ~70–75D |

:::tip Не одне і те ж
Shore A 95 та Shore D 40 — не одне і те ж, навіть якщо числа здаються близькими. Шкали різні та лише частково перекриваються в районі межі 100A/55D. Завжди перевіряйте, яку шкалу вказує постачальник.
:::

## Температурні допуски — HDT та VST

Знати, при якій температурі матеріал починає деформуватися, критично для функціональних деталей. Використовуються два стандартних вимірювання:

- **HDT (Heat Deflection Temperature)** — температура, при якій матеріал прогинається на 0,25 мм під стандартизованим навантаженням. Міра робочої температури під навантаженням.
- **VST (Vicat Softening Temperature)** — температура, при якій стандартизована голка занурюється на 1 мм у матеріал. Міра абсолютної точки розм'якшення без навантаження.

| Матеріал | HDT (°C) | VST (°C) | Практична макс. температура |
|---|---|---|---|
| PLA | 52–60 | 55–65 | ~50 °C |
| PETG | 70–80 | 75–85 | ~70 °C |
| ABS | 85–105 | 95–110 | ~90 °C |
| ASA | 90–105 | 95–108 | ~90 °C |
| TPU | 40–70 | різниться | ~60 °C |
| PA (Нейлон) | 70–180 | 180–220 | ~80–160 °C |
| PA-CF | 100–200 | 200–230 | ~100–180 °C |
| PC | 120–140 | 145–160 | ~120 °C |
| PLA-CF | 55–65 | 60–70 | ~55 °C |

:::warning PLA у гарячому середовищі
Деталі з PLA у автомобілі влітку — рецепт катастрофи. Панель приладів у припаркованому автомобілі може досягати 80–90 °C. Використовуйте ABS, ASA або PETG для всього, що може перебувати на сонці або в теплі.
:::

:::info Варіанти PA мають дуже різні властивості
PA — це сімейство матеріалів, а не один матеріал. PA6 має нижчий HDT (~70 °C), тоді як PA12 та PA6-CF можуть перебувати в діапазоні 160–200 °C. Завжди перевіряйте технічний лист для конкретного філаменту, що використовуєте.
:::

## Вимоги до сопла

### Латунне сопло (стандартне)

Підходить для всіх матеріалів **без** наповнювача з вуглецевого або скляного волокна:
- PLA, PETG, ABS, ASA, TPU, PA, PC, PVA
- Латунь м'яка та швидко зношується абразивними матеріалами

### Сопло із загартованої сталі (необхідне для композитів)

**НЕОБХІДНЕ** для:
- PLA-CF (PLA з вуглецевого волокна)
- PETG-CF
- PA-CF
- ABS-GF (ABS зі скляного волокна)
- PPA-CF, PPA-GF
- Усі філаменти з «-CF», «-GF», «-HF» або «вуглецеве волокно» у назві

Загартована сталь має нижчу теплопровідність, ніж латунь — компенсуйте +5–10 °C на температурі сопла.

:::danger Філаменти з вуглецевого волокна швидко псують латунні сопла
Латунне сопло може помітно зноситися після всього кількох сотень грамів CF-філаменту. Результат — поступова недоекструзія та неточні розміри. Інвестуйте в загартовану сталь, якщо друкуєте композити.
:::
