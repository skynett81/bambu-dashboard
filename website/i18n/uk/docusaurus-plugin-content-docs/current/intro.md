# Ласкаво просимо до Bambu Dashboard

**Bambu Dashboard** — це самостійно розміщена, повнофункціональна панель управління для 3D-принтерів Bambu Lab. Вона надає вам повний контроль над принтером, запасами філаментів, історією друку та багатьом іншим — усе з одної вкладки браузера.

## Що таке Bambu Dashboard?

Bambu Dashboard підключається безпосередньо до вашого принтера через MQTT по LAN, без залежності від серверів Bambu Lab. Ви також можете підключитися до Bambu Cloud для синхронізації моделей та історії друку.

### Основні функції

- **Живий dashboard** — температура в реальному часі, прогрес, камера, статус AMS
- **Запаси філаментів** — відстеження всіх котушок, кольорів, синхронізація AMS, сушіння
- **Історія друку** — повний журнал зі статистикою та експортом
- **Планувальник** — перегляд календаря та черга друку
- **Керування принтером** — температура, швидкість, вентилятори, консоль G-code
- **Сповіщення** — 7 каналів (Telegram, Discord, електронна пошта, ntfy, Pushover, SMS, webhook)
- **Кілька принтерів** — підтримує всю лінійку Bambu Lab: X1C, X1E, P1S, P1P, P2S, A1, A1 mini, A1 Combo, H2S, H2D, H2C та інші
- **Самостійно розміщений** — ніякої залежності від хмари, ваші дані на вашій машині

Ви можете спробувати dashboard без фізичного принтера, запустивши `npm run demo`. Це запускає 3 симульовані принтери з живими циклами друку.
:::

## Підтримувані принтери

- **Серія X1**: X1C, X1C Combo, X1E
- **Серія P1**: P1S, P1S Combo, P1P
- **Серія P2**: P2S, P2S Combo
- **Серія A**: A1, A1 Combo, A1 mini
- **Серія H2**: H2S, H2D (подвійне сопло), H2C (змінювач інструментів, 6 голівок)

## Технічний огляд

Bambu Dashboard побудований на Node.js 22 та ванільному HTML/CSS/JS — ніяких важких фреймворків, ніяких кроків збірки. База даних — SQLite, вбудована в Node.js 22. Дивіться [Архітектуру](./avansert/arkitektur) для деталей.
