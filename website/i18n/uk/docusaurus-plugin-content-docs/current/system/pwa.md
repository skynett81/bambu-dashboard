# PWA (Progressive Web App)

Bambu Dashboard можна встановити як Progressive Web App (PWA) — досвід, схожий на додаток, безпосередньо з браузера без магазину додатків. Ви отримаєте швидший доступ, push-сповіщення у фоновому режимі та обмежену офлайн-функціональність.

## Встановлення як додаток

### Робочий стіл (Chrome / Edge / Chromium)

1. Відкрийте `https://localhost:3443` у браузері
2. Знайдіть значок **Встановити** в адресному рядку (стрілка вниз зі значком екрана)
3. Натисніть на нього
4. Натисніть **Встановити** у діалозі
5. Bambu Dashboard відкривається як окреме вікно без браузерного інтерфейсу

Альтернативно: натисніть на три крапки (⋮) → **Встановити Bambu Dashboard...**

### Робочий стіл (Firefox)

Firefox не підтримує повне встановлення PWA безпосередньо. Використовуйте Chrome або Edge для найкращого досвіду.

### Мобільний (Android – Chrome)

1. Відкрийте **https://ip-вашого-сервера:3443** у Chrome
2. Натисніть на три крапки → **Додати на головний екран**
3. Дайте додатку назву та натисніть **Додати**
4. Значок з'являється на головному екрані — додаток відкривається на весь екран без браузерного інтерфейсу

### Мобільний (iOS – Safari)

1. Відкрийте **https://ip-вашого-сервера:3443** у Safari
2. Натисніть значок **Поділитися** (квадрат зі стрілкою вгору)
3. Прокрутіть вниз та виберіть **Додати на головний екран**
4. Натисніть **Додати**

:::warning Обмеження iOS
iOS має обмежену підтримку PWA. Push-сповіщення працюють лише на iOS 16.4 та новіших. Офлайн-режим обмежений.
:::

## Офлайн-режим

PWA кешує необхідні ресурси для обмеженого офлайн-використання:

| Функція | Доступно офлайн |
|---|---|
| Останній відомий статус принтера | ✅ (з кешу) |
| Історія друку | ✅ (з кешу) |
| Сховище філаменту | ✅ (з кешу) |
| Статус у реальному часі (MQTT) | ❌ Потрібне з'єднання |
| Потік камери | ❌ Потрібне з'єднання |
| Надсилання команд принтеру | ❌ Потрібне з'єднання |

Офлайн-вигляд показує банер вгорі: «З'єднання втрачено — відображаються останні відомі дані».

## Push-сповіщення у фоновому режимі

PWA може надсилати push-сповіщення, навіть якщо додаток не відкрито:

1. Відкрийте PWA
2. Перейдіть до **Налаштування → Сповіщення → Browser Push**
3. Натисніть **Увімкнути push-сповіщення**
4. Прийміть діалог дозволів
5. Сповіщення доставляються до центру сповіщень операційної системи

Push-сповіщення працюють для всіх подій, налаштованих у [Сповіщення](../funksjoner/notifications).

:::info Service Worker
Push-сповіщення вимагають, щоб браузер працював у фоновому режимі (без повного вимкнення системи). PWA використовує Service Worker для отримання.
:::

## Значок додатку та вигляд

PWA автоматично використовує значок Bambu Dashboard. Для налаштування:

1. Перейдіть до **Налаштування → Система → PWA**
2. Завантажте власний значок (мінімум 512×512 пікселів PNG)
3. Вкажіть **Назву додатку** та **Коротку назву** (відображається під значком на мобільному)
4. Виберіть **Колір теми** для рядка стану на мобільному

## Оновлення PWA

PWA оновлюється автоматично при оновленні сервера:

- Відображається ненав'язливий банер: «Доступна нова версія — натисніть для оновлення»
- Натисніть на банер для завантаження нової версії
- Повторне встановлення вручну не потрібне
