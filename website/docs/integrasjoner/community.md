# Fellesskaps-filamenter

Fellesskaps-filamenter er en delt database med filamentprofiler bidratt av Bambu Dashboard-brukere fra hele verden. Finn de beste innstillingene for ditt filament, og del dine egne erfaringer.

Gå til: **https://localhost:3443/#community**

## Bla i fellesskapsprofilene

### Søk

1. Gå til **Fellesskaps-filamenter**
2. Bruk søkefeltet for å finne etter:
   - Materiale (PLA, PETG, ABS, PA, TPU, osv.)
   - Leverandør (PolyMaker, Bambu Lab, Prusament, Sunlu, osv.)
   - Fargenavn eller farge-hex
   - Egenskap (gjennomsiktig, matt, silke, glitter, osv.)
3. Filtrer på **Rating** (kun høyest rangerte)
4. Filtrer på **Printermodell** for å se profiler optimert for din printer

### Profilkort

Hvert profilkort viser:

| Felt | Eksempel |
|---|---|
| Filamentnavn | PolyTerra PLA – Matte Forest Green |
| Materiale | PLA |
| Leverandør | PolyMaker |
| Bidragsyter | brukernavn |
| Rating | ⭐⭐⭐⭐½ (4.6 / 5) |
| Antall tester | 23 |
| Anbefalte temperaturer | Dyse: 215°C / Seng: 55°C |
| Anbefalt hastighet | Standard |

Klikk på kortet for å se fullstendig profil med alle parametre og brukerkommentarer.

## Importere en profil

1. Åpne en profil fra fellesskapet
2. Klikk **Importer til lager**
3. Velg om du vil:
   - **Koble til eksisterende spole** — profilen kobles til en registrert spole
   - **Opprett ny spole** — oppretter en ny sporpost med profildata
4. Juster temperatur eller andre innstillinger etter behov
5. Klikk **Importer**

Profilen er nå tilgjengelig i filamentlageret ditt.

## Bidra med egne profiler

### Eksport og deling

1. Gå til **Filamentlager**
2. Finn spolen du vil dele
3. Klikk **Del med fellesskapet**
4. Fyll inn:
   - **Leverandørnavn** — nøyaktig leverandørnavn
   - **Produktnavn** — eksakt produktnavn fra emballasjen
   - **Fargenavn** — offisielt fargenavn og farge-hex
   - **Kommentar** — dine erfaringer, tips og triks
   - **Printermodell** — hvilken printer du testet på
5. Klikk **Send inn**

Profilen er under vurdering og publiseres vanligvis innen 24 timer.

### Retningslinjer for bidrag

:::warning Kvalitetskrav
- Oppgi kun reelle, testede innstillinger — ikke spekulative verdier
- Bruk eksakt leverandør- og produktnavn (sjekk emballasjen)
- Én profil per spesifikt produkt og printermodell
- Unngå duplikatprofiler — sjekk om profilen allerede finnes
:::

## Vurdere profiler

Hjelp fellesskapet ved å vurdere profiler du har testet:

1. Åpne en fellesskapsprofil
2. Klikk **Gi vurdering**
3. Angi antall stjerner (1–5) og en kort kommentar
4. Klikk **Send**

Profiler med høy rating vises øverst i søkeresultater.

## Offline-bruk

Fellesskapsbasen synkroniseres lokalt automatisk (oppdateres daglig). Alle profiler er tilgjengelige uten internettforbindelse etter første sync.

Last ned basen manuelt: **Fellesskaps-filamenter → Innstillinger → Synkroniser nå**
