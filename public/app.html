Android Installation

iPhone / iPad — Option A: Native App (IPA)

  1. Download 3DPrintForge.ipa from the button above
  2. Install using one of these methods:
    • AltStore — install via altstore.io (free, re-signs every 7 days)
    • Sideloadly — drag IPA into sideloadly.io app on PC/Mac
    • Apple Configurator — for managed devices (Mac only)
    • TestFlight — if you have an Apple Developer account
  3. Open the app and enter your server URL: https://...

iPhone / iPad — Option B: PWA (no download needed)

  1. Open https://... in Safari
  2. Tap the Share button ↑ at the bottom
  3. Tap "Add to Home Screen" → Add
  4. Opens fullscreen with its own app icon
Tip: PWA supports push notifications on iOS 16.4+, works offline. The native IPA gives better performance and full notification support.
